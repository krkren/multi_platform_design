Window.setMinSize

Function/Meaning

[Windows*] Sets the minimum window size

Type
Method of the Window class
Syntax
setMinSize(width, height)
Arguments
width

Specifies the minimum width of the window.
If 0 is specified, there is no limit.

height

Specifies the minimum height of the window.
If 0 is specified, there is no limit.

Return Value
None (void)
Description

Specifies the minimum size of the window.
The window cannot be made smaller than the size specified by this method.
