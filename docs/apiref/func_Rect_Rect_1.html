Rect.Rect

Function/Meaning

Constructs a Rect object

Type
Method of Rect class
Syntax
Rect(left, top, right, bottom)
Arguments
left

Specifies the left edge.

top

Specifies the top edge.

right

Specifies the right edge.

bottom

Specifies the bottom edge.

Return Value
None (void)
Description

Constructs a Rect class object with the specified dimensions.
