Layer.getMaskPixel

Function/Meaning

Get the value of the mask image

Type
Method of Layer class
Syntax
getMaskPixel(x, y)
Arguments
x

Specifies the x-coordinate (in layer image coordinates) from which to get the value.

y

Specifies the y-coordinate (in layer image coordinates) from which to get the value.

Return Value

Mask image value (0 - 255)

Description

Gets the value (0 - 255) at an arbitrary position in the layer mask image (the image holding opacity information).
An exception will occur if a position invalid as image coordinates (out of range) is specified.
