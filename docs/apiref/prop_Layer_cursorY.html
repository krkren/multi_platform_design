Layer.cursorY

Function/Meaning
Mouse cursor y-position
Type
Property of the Layer class
Description

Represents the y-coordinate value of the mouse cursor on the layer, in pixels relative to the display coordinates.
The value can also be set.
When setting the value, the mouse cursor does not move just by setting the cursorX property.
The mouse cursor moves when the cursorY property is subsequently set.
