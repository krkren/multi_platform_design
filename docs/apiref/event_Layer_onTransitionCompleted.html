Layer.onTransitionCompleted

Function/Meaning

Transition completed

Type
Event of the Layer class
Syntax
onTransitionCompleted(dest, src)
Arguments
dest

The destination layer object of the transition (usually this) is passed.

src

The source layer object of the transition is passed.
This may be null depending on the type of transition.

Return Value
None (void)
Description

Called when a transition has completed.
