Debug.notice

Function/Meaning

Output an important message to the console

Type
Method of the Debug class
Syntax
notice(message)
Arguments
message

Specifies the message to output.

Return Value
None (void)
Description

Outputs a message to the console.
Unlike Debug.message, messages output here will always be written to the log file, even if log file writing is started midway (messages output with Debug.message are only written back to a certain extent).
Use this to output important information that will be useful when collecting log files.
