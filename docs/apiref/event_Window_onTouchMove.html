Window.onTouchMove

Function/Meaning

Finger moved

Type
Window class event
Syntax
onTouchMove(x, y, cx, cy, id)
Arguments
x

The x-coordinate of the touch position (in client coordinates).

y

The y-coordinate of the touch position (in client coordinates).

cx

The number of horizontal pixels the finger is in contact with.
Always 1 if the device does not support this.

cy

The number of vertical pixels the finger is in contact with.
Always 1 if the device does not support this.

id

Touch ID.
During multi-touch, a unique value is assigned to each position, allowing identification via this ID.

Return Value
None (void)
Description

Occurs when a finger touching the touch panel moves.
