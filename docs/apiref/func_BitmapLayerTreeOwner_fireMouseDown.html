BitmapLayerTreeOwner.fireMouseDown

Function/Meaning

Notifies the layer of a mouse down event

Type
Method of the BitmapLayerTreeOwner class
Syntax
fireMouseDown(x, y, mb, flags)
Arguments
x

The x-coordinate where the mouse button was pressed (client coordinates)

y

The y-coordinate where the mouse button was pressed (client coordinates)

mb

The mouse button that was pressed

flag

The state of modifier keys (Shift, etc.) held down when the mouse button was pressed

Return Value
None (void)
Description
