Window.focusable

Function/Meaning
[Windows*] Whether focus can be obtained
Type
Property of the Window class
Description

Represents whether focus can be obtained.
The value can also be set.
If set to false, focus cannot be obtained; in other words, the window will not become active.
As a side effect, operations such as moving the window by grabbing the title bar, resizing the window, or closing the window by pressing the "X" button will also be disabled.
If you want to receive keyboard input, you can use the Window.trapKey property.
Normally, this is used for implementing windows like popup menus that are displayed in the foreground but do not gain focus themselves.
In the current version, the settings of this property may be lost when the window enters or exits full-screen mode.
On Android, this is always true.
