Array.pack

Function/Meaning

Converts array elements into binary (Octet) according to the specified string.

Type
Array class method
Syntax
pack(template:string)
Arguments
template

The following characters can be used in the template string.

  • a : ASCII string (padded with null characters)
  • A : ASCII string (padded with spaces)
  • b : Binary string (low bit to high bit order)
  • B : Binary string (high bit to low bit order)
  • c : Signed 1-byte integer (-128 to 127)
  • C : Unsigned 1-byte integer (0 to 255)
  • d : Double-precision floating-point value
  • f : Single-precision floating-point value
  • h : Hexadecimal string (low nybble first)
  • H : Hexadecimal string (high nybble first)
  • i : Signed int value (usually 4 bytes)
  • I : Unsigned int value (usually 4 bytes)
  • l : Signed long value (usually 4 bytes)
  • L : Unsigned long value (usually 4 bytes)
  • n : Unsigned short value (network byte order)
  • N : Unsigned long value (network byte order)
  • s : Signed short value (usually 2 bytes)
  • S : Unsigned short value (usually 2 bytes)
  • v : Unsigned short value (little endian)
  • V : Unsigned long value (little endian)
  • x : Null character
  • X : Back up one byte
  • @ : Fill with null characters to absolute position
  • m : Base64 decode
Return Value
None (void)
Description

The specified string is called a template.
A template consists of a sequence of type specification characters and their lengths.
When * is specified as the length, it represents all remaining length.
If the length is omitted, it is treated as 1.
