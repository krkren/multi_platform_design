Debug.logAsError

Function/Meaning

Start logging to a file as if an error occurred

Type
Debug class method
Syntax
logAsError()
Arguments
Return Value
None (void)
Description

Regarding the error log file, this makes KiriKiri behave the same way as when an error occurs.
Specifically, if Debug.logToFileOnError is true, it starts outputting the console log to a file.
At that time, if Debug.clearLogFileOnError is true, the file is cleared.
In contrast, Debug.startLogToFile starts outputting the console log to a file unconditionally.

Reference
Debug.startLogToFile
Debug.logToFileOnError
Debug.clearLogFileOnError
