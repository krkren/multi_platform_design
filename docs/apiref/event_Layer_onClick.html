Layer.onClick

Function/Meaning

Layer was clicked

Type
Event of the Layer class
Syntax
onClick(x, y)
Arguments
x

The x-coordinate (in layer display coordinates) where the layer was clicked.

y

The y-coordinate (in layer display coordinates) where the layer was clicked.

Return Value
None (void)
Description

Occurs when the layer is clicked.

Reference
Layer.onMouseDown
Layer.onDoubleClick
