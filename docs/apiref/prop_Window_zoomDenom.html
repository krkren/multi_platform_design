Window.zoomDenom

Function/Meaning
[Windows*] Layer zoom factor (denominator)
Type
Property of the Window class
Description

Represents the denominator of the layer zoom factor.
Although you can set the value, please use the Window.setZoom method when setting values.
The numerator is represented by the Window.zoomNumer property.
For details, please refer to the Window.setZoom method.

Reference
Window.setZoom
Window.zoomNumer
