Window.onPointerMove

Function/Meaning

A pointing device was moved

Type
Window class event
Syntax
onPointerMove(type, x, y, cx, cy, shift, id)
Arguments
type

Touch or mouse. Notified as mouse, not by mouse button type.

x

X position

y

Y position

cx

Contact width (pixels), 1 for mouse

cy

Contact height (pixels), 1 for mouse

shift

Modifier key state (including mouse button state), 0 for touch

id

Touch ID, always 0 for mouse

Return Value
None (void)
