String.substring

Function/Meaning

Returns a substring of the string

Type
Method of the String class
Syntax
substring(offset:int, count:int=void) :string
Arguments
offset

Starting character position

count

Number of characters, optional.

Return Value

Substring

Description

Returns a substring of the string from offset for count characters.
Does not affect the original string.
If count is omitted, all characters from offset onwards are returned.
Note that the meaning of the arguments differs from the JavaScript method of the same name.
