Rect.clip

Function/Meaning

Rectangle clipping

Type
Method of Rect class
Syntax
clip(r)
Arguments
r

Specifies the rectangle object to clip with.

Return Value

Whether clipping was performed

Description

Clips the rectangle with the specified rectangle.
Clipping is performed only if the resulting rectangle is not empty.
If it would become empty, false is returned and clipping is not performed.
