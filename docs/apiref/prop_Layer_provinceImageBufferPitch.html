Layer.provinceImageBufferPitch

Function/Meaning
Province image buffer pitch
Type
Property of the Layer class
Description

Represents the pitch (the number of bytes to the next scanline) of the province image buffer (an 8bpp bitmap containing province information).
This property exists to provide a means of direct access to the image buffer for plugins and other purposes.
Assuming tjs_uint8 is an 8-bit (1-byte) integer type, the pointer to the image position (x, y) can be calculated in C as follows:
(tjs_uint8*)provinceImageBuffer + y * provinceImageBufferPitch + x
The value of this property may be slightly larger than the number of bytes exactly required for the image width.
Note that this property value can be negative.
