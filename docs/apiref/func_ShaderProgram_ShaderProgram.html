ShaderProgram.ShaderProgram

Function/Meaning

Constructor

Type
Method of the ShaderProgram class
Syntax
ShaderProgram(vertex:string, fragment:string, vsfile:boolean=true, fsfile:boolean=true)
Arguments
vertex

Specifies the vertex shader.
When vsfile is true, specify the filename; when false, specify the string directly.

fragment

Specifies the fragment shader.
When fsfile is true, specify the filename; when false, specify the string directly.

vsfile

Selects whether the string specified in vertex is a file or a direct script specification.

fsfile

Selects whether the string specified in fragment is a file or a direct script specification.

Return Value
None (void)
Description
