Layer.beginTransition

Function/Meaning

Start transition

Type
Layer class method
Syntax
beginTransition(name, withchildren=true, transsrc=null, options)
Arguments
name

Specifies the transition name.
By default, 'crossfade', 'universal' (universal transition), and 'scroll' (scroll transition) are defined.

withchildren

Whether to perform the transition including child layers.
If true, the transition includes child layers.
If false, only the layer executing the method transitions.

transsrc

Specifies the layer that will be the transition destination.
Depending on the transition, there may be cases where this specification is not necessary (such as transitions that occur independently).

options

Specifies transition options as a dictionary array.
Required options vary depending on the transition type.
Regardless of the transition type, 'selfupdate' and 'callback' members can be specified as common options.
If the 'selfupdate' member is true, KiriKiri will not perform automatic screen updates.
In this case, the screen must be updated appropriately on the program side using methods like Layer.update.
This feature is useful when you want to perfectly synchronize transition rendering with program-driven rendering.
If a TJS2 method is specified in the 'callback' member, that method is called every time rendering actually occurs.
In this method (the callback method), please return a 'tick' value of 0 or greater.
Many transitions operate based on a 'tick' value representing actual time in milliseconds. By implementing this callback method to return an arbitrary tick, you can rewind the transition effect to any point or fast-forward it, allowing control over the transition regardless of actual time.
Many transitions consider the value specified in the 'time' member of the options to be the end of the transition.
Therefore, for example, by specifying a value like 1000 for 'time' and having the callback method return a value less than 1000, you can display the transition at any specific stage.
Also, in many transitions, the transition stops when the 'tick' value reaches the value specified in 'time'. If you do not want the transition to stop, ensure that you always specify a value less than the value specified in 'time'.

Return Value
None (void)
Description

Starts the transition.
When children=true, the parent-child tree structure is swapped with the transition source specified by transsrc upon completion.
When children=false, only the transition source layer and the layer that executed the method (the transition destination layer) are swapped.
Please note that in both cases, the swap occurs within the tree structure.
This method returns immediately after starting the transition.
