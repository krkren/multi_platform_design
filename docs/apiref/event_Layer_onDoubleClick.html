Layer.onDoubleClick

Function/Meaning

Layer was double-clicked

Type
Event of the Layer class
Syntax
onDoubleClick(x, y)
Arguments
x

The x-coordinate where the layer was double-clicked (in layer display coordinates).

y

The y-coordinate where the layer was double-clicked (in layer display coordinates).

Return Value
None (void)
Description

Occurs when the layer is double-clicked.

Reference
Layer.onClick
