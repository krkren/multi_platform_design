VideoOverlay.setMixingLayer

Function/Meaning

[Windows+] Set mixing target layer

Type
VideoOverlay class method
Syntax
setMixingLayer(layer)
Arguments
layer

Layer

Return Value
None (void)
Description

Mixes the video with the specified layer.
Mixing is performed with the layer image as it exists at the moment this method is called.
To reflect updates to the layer image, you must call this method again.
By using VideoOverlay.onFrameUpdate to check if the layer image has been updated and then calling this method, updates can be reflected automatically.
Calling this method every frame in onFrameUpdate will result in a significant processing load, so it should be avoided.
Available only in mixer mode.
