Layer.releaseCapture

Function/Meaning

Release mouse event capture

Type
Layer class method
Syntax
releaseCapture()
Arguments
Return Value
None (void)
Description

Releases the mouse event capture.
Mouse event capture is a feature where mouse events are sent exclusively to the layer where the mouse button was first pressed until the button is released.
This method releases this feature and returns to the normal mouse event processing state.
When this method is executed, the mouse capture for layers belonging to the same window is released, even if the layer executing the method is different from the layer being captured.
This method does nothing if not in a captured state.
