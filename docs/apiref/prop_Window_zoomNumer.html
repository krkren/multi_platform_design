Window.zoomNumer

Function/Meaning
[Windows*] Layer zoom factor (numerator)
Type
Property of Window class
Description

Represents the numerator of the layer zoom factor.
Although you can set the value, please use the Window.setZoom method when setting values.
The denominator is represented by the Window.zoomDenom property.
See the Window.setZoom method for details.

Reference
Window.setZoom
Window.zoomDenom
