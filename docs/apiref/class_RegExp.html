RegExp

The RegExp class handles regular expression patterns.
Perl-compatible regular expression patterns can be used.
It is similar to the JavaScript RegExp class, but compatibility is limited.

Copyright Notice

The regular expression implementation uses K.Kosako's Oniguruma library.
We thank the author for publishing this useful library.

Regular Expression Patterns

Regular expression patterns are specified between / and / delimiters.
See Token.
