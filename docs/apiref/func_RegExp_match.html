RegExp.match

Function/Meaning

Performs regular expression pattern matching on the specified string and returns an array containing the match results

Type
Method of the RegExp class
Syntax
match(text:string) :Array
Arguments
text

String

Return Value

If the pattern does not match, the number of elements in the array will be 0.
If it matches, element 0 (the first element) contains the entire matched part, and subsequent elements contain each captured group (parts specified by ( ) in the regular expression pattern).

Description

This is identical to the exec method, except that it does not update the property values of this object.
