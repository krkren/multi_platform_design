ImageFunction.fillRect

Function/Meaning

Fill Rectangle

Type
Method of ImageFunction class
Syntax
fillRect(bmp, value, rect=null, isalpha=true, cliprect=null)
Arguments
bmp

Specifies the Bitmap object to be filled.

value

Specifies the color or value to fill with.
The meaning of this value changes depending on the value of isalpha.

  • true : Specify opacity and color in 0xAARRGGBB format. Both main and mask will be filled.
  • false : Specify color in 0xRRGGBB format.
rect

Specifies the rectangle to fill as a Rect object in pixels (relative to the image position).
If omitted, the entire area is targeted.

isalpha

Specifies whether it has an alpha channel.

cliprect

Specifies the clipping region as a Rect object (relative to the Bitmap image position).
If omitted, the entire area is targeted.

Return Value
None (void)
Description

Fills the specified rectangle of the specified Bitmap image using the specified method.
