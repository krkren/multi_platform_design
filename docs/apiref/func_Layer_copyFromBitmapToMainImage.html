Layer.copyFromBitmapToMainImage

Function/Meaning

Copy image from Bitmap

Type
Layer class method
Syntax
copyFromBitmapToMainImage(src)
Arguments
src

Specifies the source Bitmap object.

Return Value
None (void)
Description

Copies the image from the Bitmap object specified by src to the layer's main image.
The image size will be the same as the image size of the source Bitmap object.
Although it is called a copy, in reality, it simply results in a state where "the same image is shared by two or more layers," so this method takes almost no execution time.
