BitmapLayerTreeOwner.fireTouchMove

Function/Meaning

Notifies the layer that a touch has moved

Type
BitmapLayerTreeOwner class method
Syntax
fireTouchMove(x, y, cx, cy, id)
Arguments
x

x-coordinate of the touch position (client coordinates)

y

y-coordinate of the touch position (client coordinates)

cx

Number of horizontal pixels in contact

cy

Number of vertical pixels in contact

id

Touch ID

Return Value
None (void)
Description
