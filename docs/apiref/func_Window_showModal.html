Window.showModal

Function/Meaning

[Windows+] Display window modally

Type
Method of Window class
Syntax
showModal()
Arguments
Return Value
None (void)
Description

Displays the window modally (with mode).
This method makes the window visible and temporarily disables other windows.
Therefore, the user can only access this window (this is called a modal state).
When the window is closed, the modal state is released.
The window must be hidden at the time this method is called.
