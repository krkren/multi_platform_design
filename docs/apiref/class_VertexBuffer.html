VertexBuffer

VertexBuffer is a class that holds vertex information.

It is used when you want to set the position, UV coordinates, vertex color, etc. in detail.
It is also used when you want to perform mesh deformation instead of a simple rectangle, or when you want to use point sprites.

Pointer access is also possible so that the vertex array can be directly changed by the plugin (however, OpenGLES3.0 or later).
Traditional Layer extensibility can be ensured by vertex manipulation using shader + TJS2 script + plugin.
It is a little more complicated than what was almost completed in the conventional plugin.
