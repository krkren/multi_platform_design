Layer.neutralColor

Function/Meaning
Neutral color
Type
Property of the Layer class
Description

Represents the neutral color of the layer in 0xAARRGGBB format.
A value can also be set.
The layer's neutral color is set to the neutral color of that type when the Layer.type property is changed.
The neutral color becomes the initial value used to fill the expanded area when the layer image size is expanded.
By setting a value, you can specify the initial color used when expanding the layer image size.
