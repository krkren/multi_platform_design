VideoOverlay.height

Function/Meaning
[Windows*] Height of the playback rectangle
Type
VideoOverlay class property
Description

Represents the height of the playback rectangle.
You can also set the value.
This property cannot be used during layer playback in the current version (the size is always the same as the video size).
On Android, it always returns 0, and setting it has no effect.
