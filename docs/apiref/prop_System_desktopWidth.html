System.desktopWidth

Function/Meaning
Desktop width
Type
Property of the System class
Description

Represents the width of the desktop (the area where windows can be displayed) in pixels.
The value is for the display where the main window is located.
If there is no main window, the primary display is targeted.
On Android, this is the screen width.
