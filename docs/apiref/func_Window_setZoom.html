Window.setZoom

Function/Meaning

[Windows*] Set layer zoom ratio

Type
Method of the Window class
Syntax
setZoom(numer, denom)
Arguments
numer

Specifies the numerator of the zoom ratio as an integer.

denom

Specifies the denominator of the zoom ratio as an integer.

Return Value
None (void)
Description

Specifies the zoom ratio of the layer.
The image will be displayed enlarged or reduced according to the size specified by numerator/denominator.
If the numerator and denominator have a common divisor, they are automatically reduced. Therefore, the values read from the Window.zoomNumer and Window.zoomDenom properties may differ from the values specified in this method.
In the current version of KiriKiri, whether interpolation is applied during scaling depends on the graphics card hardware and drivers.
On hardware or drivers where interpolation is not applied, the image quality will be coarse.
Depending on the options, KiriKiri may check for available scaling hardware every time the ratio is changed. Therefore, this method is not suitable for purposes such as creating effects by continuously changing the zoom ratio.
This has no effect during Canvas rendering.
