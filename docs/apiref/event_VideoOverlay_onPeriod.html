VideoOverlay.onPeriod

Function/Meaning

[Windows*] A Period event occurred

Type
VideoOverlay class event
Syntax
onPeriod(type)
Arguments
type

Represents the type of Period event.
It is one of the following:

  • perLoop : Reached the end of a (normal) loop
  • perSegLoop : Reached the end of a segment loop
  • perPeriod : Reached the frame specified by the setPeriodEvent method
  • perPrepare : Playback preparation by the prepare method is complete
Return Value
None (void)
Description

This method is called when the end of a loop is reached, when a frame specified by setPeriodEvent is reached, or when playback preparation is completed by the prepare method.
When this event is called upon reaching the end of a loop or a frame specified by setPeriodEvent, the playback position may have already passed the specified position depending on the playback status.
To get the actual current playback position, refer to the frame property.
Does not occur on Android.
