BasicDrawDevice

The Window.BasicDrawDevice class is a DrawDevice for registering this instance in Window.drawDevice and using it, and provides standard drawing functions.
Available only on Windows.
It is deprecated in the multi-platform version.
Drawing is done using Canvas by default, and DrawDevice cannot be used unless DrawDevice is specified with -graphicsystem=drawdevice as an option.
In that case, drawing with Canvas will no longer be possible.
