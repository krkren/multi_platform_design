Rect.includedPos

Function/Meaning

Rectangle inclusion check

Type
Method of Rect class
Syntax
includedPos(x, y)
Arguments
x

Specifies the X-axis coordinate value to check.

y

Specifies the Y-axis coordinate value to check.

Return Value

Whether it is included or not

Description

Determines if the specified position is included within the rectangle.
Returns true if it is included.
The range is determined excluding right and bottom.
