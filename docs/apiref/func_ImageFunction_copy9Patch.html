ImageFunction.copy9Patch

Function/Meaning

Image copy using 9-patch (Multi-platform version and later)

Type
ImageFunction class method
Syntax
copy9Patch(dst:Bitmap, src:Bitmap) :Rect
Arguments
dst

Destination Bitmap

src

Source Bitmap

Return Value

Margin information

Description

Performs an image copy using 9-patch (slice).
The returned margin information is a Rect class object.
Draw text or other elements within this margin.
