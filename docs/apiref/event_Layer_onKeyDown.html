Layer.onKeyDown

Function/Meaning

Key pressed

Type
Event of the Layer class
Syntax
onKeyDown(key, shift, process=true)
Arguments
key

The virtual key code value of the pressed key.

shift

The state of the shift keys and mouse buttons that were pressed at the same time the key was pressed.
It is a combination of the following values using bitwise OR.

  • ssAlt : ALT key was pressed
  • ssShift : SHIFT key was pressed
  • ssCtrl : CTRL key was pressed
  • ssLeft : Left mouse button was pressed
  • ssMiddle : Middle mouse button was pressed
  • ssRight : Right mouse button was pressed

Additionally, if the keyboard is held down for a long time and key repeat occurs, the following value is also combined.

  • ssRepeat : Key repeat occurred
process

When calling the parent class's onKeyDown, if false is passed for this argument, the parent class will not perform the default processing (such as moving focus) for that key.

Return Value
None (void)
Description

Occurs when a key is pressed.
