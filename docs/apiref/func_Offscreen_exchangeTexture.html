Offscreen.exchangeTexture

Function/Meaning

Swaps the texture set in the framebuffer with the texture pointed to by the Texture class.

Type
Offscreen class method
Syntax
exchangeTexture(texture:Texture)
Arguments
texture

The instance of the Texture class to exchange.

Return Value
None (void)
Description

An exception will occur if the sizes of the textures to be exchanged do not match.
The color format must also be RGBA.
It might be better not to perform the exchange during rendering (needs verification).
If there is no need to draw to both during a crossfade or similar, overhead can be reduced by extracting the previous image as a Texture and replacing it with a new Texture.
