WaveSoundBuffer.open

Function/Meaning

Opens media

Type
Method of the WaveSoundBuffer class
Syntax
open(storage)
Arguments
storage

Specifies the storage to be played.

Return Value
None (void)
Description

Opens the specified media.
This method does not start playback.
If a file exists with the specified storage name plus the .sli extension, it will be loaded as sound loop information.
