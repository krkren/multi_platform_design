ImageFunction.operateRect

Function/Meaning

Rectangular Operation Composition

Type
Method of ImageFunction class
Syntax
operateRect(dst, dleft, dtop, src, srcrect=null, cliprect=null, mode=omAuto, face=dfAlpha, opa=255, hda=false)
Arguments
dst

Specifies the destination Bitmap object.

dleft

Specifies the left edge position of the destination rectangle in pixels (relative to the destination Bitmap's image position).

dtop

Specifies the top edge position of the destination rectangle in pixels (relative to the destination Bitmap's image position).

src

Specifies the source Bitmap object. If unspecified, the entire image is targeted.

srcrect

Specifies the rectangle to operate on as a Rect object in pixels (relative to the source Bitmap's image position).
If unspecified, the entire image is targeted.

cliprect

Specifies the clipping rectangle as a Rect object (relative to the destination Bitmap's image position).
If unspecified, no clipping is performed.

mode

Specifies the operation mode.

  • If omPsNormal is specified, Photoshop-compatible alpha blending is performed.
  • If omPsAdditive is specified, Photoshop-compatible Linear Dodge blending is performed.
  • If omPsSubtractive is specified, Photoshop-compatible Linear Burn blending is performed.
  • If omPsMultiplicative is specified, Photoshop-compatible Multiply blending is performed.
  • If omPsScreen is specified, Photoshop-compatible Screen blending is performed.
  • If omPsOverlay is specified, Photoshop-compatible Overlay blending is performed.
  • If omPsHardLight is specified, Photoshop-compatible Hard Light blending is performed.
  • If omPsSoftLight is specified, Photoshop-compatible Soft Light blending is performed.
  • If omPsColorDodge is specified, Photoshop-compatible Color Dodge blending is performed.
  • If omPsColorDodge5 is specified, Color Dodge blending compatible with Photoshop version 5.x and earlier is performed.
  • If omPsColorBurn is specified, Photoshop-compatible Color Burn blending is performed.
  • If omPsLighten is specified, Photoshop-compatible Lighten blending is performed.
  • If omPsDarken is specified, Photoshop-compatible Darken blending is performed.
  • If omPsDifference is specified, Photoshop-compatible Difference blending is performed.
  • If omPsDifference5 is specified, Difference blending compatible with Photoshop version 5.x and earlier is performed.
  • If omPsExclusion is specified, Photoshop-compatible Exclusion blending is performed.
  • If omAdditive is specified, additive blending is performed.
  • If omSubtractive is specified, subtractive blending is performed.
  • If omMultiplicative is specified, multiplicative blending is performed.
  • If omDodge is specified, dodge blending is performed.
  • If omDarken is specified, darken blending is performed.
  • If omLighten is specified, lighten blending is performed.
  • If omScreen is specified, screen multiplication blending is performed.
  • If omAlpha is specified, alpha blending is performed.
  • If omAddAlpha is specified, additive alpha blending is performed.
  • If omOpaque is specified, the alpha information of src is ignored, and src is always treated as completely opaque.
face

Specifies the drawing method.

  • If dfAlpha is specified, the image is treated as an image with an alpha channel and drawn.
  • If dfAddAlpha is specified, the image is treated as an image with an additive alpha channel and drawn.
  • If dfOpaque is specified, the image is treated as completely opaque and drawn.
opa

Specifies the opacity of the operation (0 to 255).

hda

Specifies whether to protect the alpha channel.

Return Value
None (void)
Description

Performs a composite operation on the specified rectangular portion of the source Bitmap onto the specified position of the destination Bitmap using the specified mode.
