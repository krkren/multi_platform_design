Font.getEscWidthX

Function/Meaning

Horizontal X-coordinate movement of characters

Type
Method of the Font class
Syntax
getEscWidthX(text)
Arguments
text

Specifies the string for which you want to obtain the horizontal X-coordinate movement.

Return Value

Returns the horizontal X-coordinate movement of the characters in pixels.

Description

Obtains the amount of movement required for the character drawing position when drawing the specified text with the current font settings.
This method returns the movement on the X-axis when moving horizontally (left) relative to the character drawing direction.
