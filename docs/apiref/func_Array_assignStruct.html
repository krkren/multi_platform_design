Array.assignStruct

Function/Meaning

Copies the array including its structure.

Type
Array class method
Syntax
assignStruct(src:Array)
Arguments
src

Source Array|Dictionary instance

Return Value
None (void)
Description

Copies the contents of another array specified by the argument exactly.
Unlike the assign method, if a member is an array or a dictionary, its contents are also copied recursively (the assign method only copies the reference).
