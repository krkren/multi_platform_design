Matrix44.setTranslate

Function/Meaning

Translation setting

Type
Method of Matrix44 class
Syntax
setTranslate(x:real, y:real, z:real=0)
Arguments
x

X coordinate value to set

y

Y coordinate value to set

z

Z coordinate value to set

Return Value
None (void)
Description

Sets the values for the XYZ translation position of the matrix.
This only sets the values; no calculations are performed with the current matrix.
