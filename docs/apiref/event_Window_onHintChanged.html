Window.onHintChanged

Function/Meaning

[Windows*] When the hint state changes

Type
Window class event
Syntax
onHintChanged(text, x, y, isshow)
Arguments
text

The string to be displayed in the hint is passed.

x

The X-coordinate for the hint display is passed.

y

The Y-coordinate for the hint display is passed.

isshow

Whether to show or hide the hint is passed.

Return Value
None (void)
Description

Represents an event function called when the hint state changes.
Displays the hint according to the show/hide status.
When displaying a hint using a layer, set Layer.hitThreshold to 256 so that mouse messages for that layer are ignored.
Also, specify true for Layer.ignoreHintSensing.

Reference
