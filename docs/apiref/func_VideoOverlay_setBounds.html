VideoOverlay.setBounds

Function/Meaning

[Windows*] Specify the position and size of the playback rectangle

Type
Method of VideoOverlay class
Syntax
setBounds(left, top, width, height)
Arguments
left

Specifies the left position of the rectangular area for media playback.

top

Specifies the top position of the rectangular area for media playback.

width

Specifies the width of the rectangular area for media playback.

height

Specifies the height of the rectangular area for media playback.

Return Value
None (void)
Description

Specifies the position and size of the rectangular area for media playback.
Coordinates are specified in pixels within the window's client area (the area where layers can be displayed).
In the current version, when playing back on a layer, only the position can be specified with this method; the size is ignored.
