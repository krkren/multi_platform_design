Window.onTouchDown

Function/Meaning

Screen touched

Type
Event of the Window class
Syntax
onTouchDown(x, y, cx, cy, id)
Arguments
x

The x-coordinate of the touched position (in client coordinates).

y

The y-coordinate of the touched position (in client coordinates).

cx

The number of horizontal pixels the finger is in contact with.
Always 1 if the device does not support this.

cy

The number of vertical pixels the finger is in contact with.
Always 1 if the device does not support this.

id

Touch ID.
During multi-touch, a unique value is assigned to each touch point, allowing identification of the position by this ID.

Return Value
None (void)
Description

Occurs when the touch panel is touched.
