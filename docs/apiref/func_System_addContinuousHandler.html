System.addContinuousHandler

Function/Meaning

Add Continuous Handler

Type
System class method
Syntax
addContinuousHandler(callback)
Arguments
callback

Specifies the function to be used as the handler.

Return Value
None (void)
Description

Registers a Continuous handler.
A Continuous handler is an event handler that is called "as frequently as possible."
If there are no other processes to perform, Kirikiri will continue to call the Continuous handler.
If other events occur, they will take priority.
However, if the calling frequency is specified by the -contfreq command line option, it will follow that setting.
