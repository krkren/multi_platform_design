System.createUUID

Function/Meaning

UUID string generation

Type
System class method
Syntax
createUUID()
Arguments
Return Value

Returns the generated UUID string in the format "e8b2a2b5-5ceb-4f75-a08b-1f1bdfdca4f1" (each alphanumeric character except hyphens is a hexadecimal digit).

Description

Generates and returns a UUID string. This method generates a 128-bit UUID (universal unique identifier) based on a random bit sequence.
The UUID generation algorithm implemented in KiriKiri generates a random bit sequence by picking up some environmental noise, but it is not recommended for use in applications requiring high security.
However, it is considered "extremely unlikely to overlap" with other UUIDs.
