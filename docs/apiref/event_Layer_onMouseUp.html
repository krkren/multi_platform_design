Layer.onMouseUp

Function/Meaning

Mouse button released

Type
Event of Layer class
Syntax
onMouseUp(x, y, button, shift)
Arguments
x

The x-coordinate where the mouse button was released (in layer display coordinates).

y

The y-coordinate where the mouse button was released (in layer display coordinates).

button

The mouse button that was released.
It will be one of the following values.

  • mbLeft : The left mouse button was released
  • mbMiddle : The middle mouse button was released
  • mbRight : The right mouse button was released
  • mbX1 : The first side mouse button was released
  • mbX2 : The second side mouse button was released
shift

The state of the shift keys that were pressed when the mouse button was released.
This is a bitwise OR combination of the following values.

  • ssAlt : The ALT key was pressed
  • ssShift : The SHIFT key was pressed
  • ssCtrl : The CTRL key was pressed
Return Value
None (void)
Description

Occurs when a mouse button is released.
