Layer.flipLR

Function/Meaning

Horizontal flip

Type
Method of Layer class
Syntax
flipLR()
Arguments
Return Value
None (void)
Description

Flips the image horizontally.
This method is not affected by the drawing clip rectangle set by the Layer.setClip method, etc. (the entire layer image is always flipped).
It is also not affected by the Layer.face property or the Layer.holdAlpha property.
