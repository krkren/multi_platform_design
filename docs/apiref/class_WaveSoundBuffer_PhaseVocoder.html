WaveSoundBuffer_PhaseVocoder

The WaveSoundBuffer.PhaseVocoder class is a filter for registering and using this instance in WaveSoundBuffer.filters, and provides the function of Phase Vocoder. With Phase Vocoder, it is possible to change the pitch (pitch shifter) while maintaining the playback speed, or change the playback speed (time shifter) while maintaining the pitch.
