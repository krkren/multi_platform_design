String.toLowerCase

Function/Meaning

Returns a string with alphabetic characters converted to lowercase

Type
Method of the String class
Syntax
toLowerCase() :string
Arguments
Return Value
None (void)
Description

Returns a string with alphabetic characters converted to lowercase.
Full-width characters are not converted.
This method does not affect the original string.
In the current version, conversion is only performed for characters with character codes less than 0x80.
