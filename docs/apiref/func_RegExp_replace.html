RegExp.replace

Function/Meaning

Replaces strings and returns the string after replacement

Type
Method of RegExp class
Syntax
replace(text:string, replace:string) :string
Arguments
text

Target string

replace

Replacement string

Return Value

Performs matching on the target string, replaces the matched parts with the replacement string, and returns the string after replacement.

Description

If a function is passed as the replacement string instead of a string, that function will be called for the replacement operation.
The function takes one argument, which is an array object representing the matched part in the target string (refer to the exec method for details on this array).
The matched part in the target string is replaced by the string returned by the function.
This method ignores the start property.
