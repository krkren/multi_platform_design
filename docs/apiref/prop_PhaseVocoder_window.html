PhaseVocoder.window

Function/Meaning
Window size
Type
PhaseVocoder class property
Description

Represents the window size (frame size) of the FFT (Fast Fourier Transform) used internally by PhaseVocoder.
Specify a power of 2 between 64 and 32768.
The default is 4096.
Increasing this value improves resolution on the frequency axis (pitch becomes more accurate) but decreases resolution on the time axis (sound attacks become duller).
Increasing the value will increase CPU usage.
Find the lowest value that maintains sound quality.
For 44.1KHz voice, a window size of around 256 is recommended.
For music, a window size of around 4096 is recommended.
Changes to this property are applied when the WaveSoundBuffer.open method is executed.
