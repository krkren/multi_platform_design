Scripts.setCallMissing

Function/Meaning

Configure missing method calls

Type
Scripts class method
Syntax
setCallMissing(obj)
Arguments
obj

The object for which the missing method will be called when a member is not found.

Return Value
None (void)
Description

Configures the system to call the missing method when a member is not found.
If a member is not found in an object registered with this method, a method is called in the format: function missing( isset, name, prop ).
isset is true for setting (assignment) and false for getting (retrieval).
name is the name of the member that was not found.
prop is the value to be assigned when setting, or the destination for the return value when getting.
If the member name is still not found even within the missing method, return a non-zero value.
Returning a non-zero value will cause a "member not found" exception to be raised.
Even if a function is not found, it is called as a "get".
Since the finalize function is also subject to this, it is recommended to define a finalize function in classes using this setting.
