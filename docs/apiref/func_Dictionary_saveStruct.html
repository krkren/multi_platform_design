Dictionary.saveStruct

Function/Meaning

Outputs structured data to a file

Type
Dictionary class method
Syntax
saveStruct(storage:string, mode:string=null) :Dictionary
Arguments
storage

The name of the file to write to

mode

Specifies the mode string when writing the file.

  • Specifying "c" applies simple encryption to the file (it can be read as-is using the load method).
  • Specifying "z" allows the file to be saved with compression (cannot be used with "c"; this can also be read as-is using the load method).
  • Specifying "o" followed by a decimal offset writes to the file starting from that byte position.
  • Specifying "b" saves the file in binary format (cannot be used with other mode specifications; this can also be read as-is using the load method).
Return Value

Returns the Dictionary class object that was output.

Description

Outputs the state of the elements to a file in a restorable state.
Each element can contain strings, real numbers, octet sequences, and integers.
Also, if an element is an object, its contents will be output to the file only if it is an Array or Dictionary class object.
Other elements will be output as void.
