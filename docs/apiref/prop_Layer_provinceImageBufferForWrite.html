Layer.provinceImageBufferForWrite

Function/Meaning
Province image buffer pointer (for writing)
Type
Layer class property
Description

Represents a pointer to the upper-left corner of the province image buffer (an 8bpp bitmap containing province information).
This property exists to provide a means of direct access to the image buffer for plugins and other purposes.
It is returned as an integer type, but should be cast to an appropriate type (such as unsigned char *) in plugins.
Unlike Layer.provinceImageBuffer, values can be written to the pointer obtained through this property.
Internally, KiriKiri shares identical images between multiple layers, but accessing this property will break that sharing.
If no image is assigned to the layer, one will be automatically assigned when this property is accessed, and the entire area will be initialized with province number 0.
The size of the image is represented by the Layer.imageWidth and Layer.imageHeight properties.
Refer to Layer.provinceImageBufferPitch for the pointer calculation method.
