Layer.absolute

Function/Meaning
Absolute position
Type
Property of Layer class
Description

Represents the stacking order among sibling layers with the same parent.
Smaller values are displayed further back.
Unlike the Layer.order property, values do not need to be continuous among siblings.
Setting a value allows you to change the order among sibling layers.
Setting a value will set the parent layer's Layer.absoluteOrderMode property to true.
