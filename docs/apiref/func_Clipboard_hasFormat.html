Clipboard.hasFormat

Function/Meaning

[Windows*] Checks if the specified format exists in the clipboard

Type
Method of the Clipboard class
Syntax
hasFormat(format)
Arguments
format

Specifies the format you want to check for.
In the current version, only cbfText can be specified. Specifying this allows you to check if text format data exists in the clipboard.

Return Value

True if the specified format exists in the clipboard, false otherwise

Description

You can check whether the specified format exists in the clipboard.
Since the clipboard provided by the OS can hold multiple formats simultaneously, you can use this method to check if a specific format is among them.
Always returns false on Android.
