Window.stayOnTop

Function/Meaning
[Windows*] Whether to always display on top
Type
Property of the Window class
Description

Indicates whether the window is always displayed on top (at the very front).
You can also set the value.
If true, the window is always displayed on top.
In the current version, the settings for this property may be lost when the window enters or exits full-screen mode.
On Android, this is always true.
