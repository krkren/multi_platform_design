VertexBuffer.VertexBuffer

Function/Meaning

Creates a vertex buffer by specifying a vertex array

Type
VertexBuffer class method
Syntax
VertexBuffer(data:Array, dataType:int, updateType:int, isIndex:bool=false)
Arguments
data

Vertex array. The buffer size is the number of elements in this array x data type size.

dataType

Data type. dtByte/dtUByte/dtShort/dtUShort/dtInt/dtFloat

updateType

Data update frequency. utStream/utStatic/utDynamic

isIndex

Whether it is an index buffer. false for a vertex buffer.

Return Value
None (void)
