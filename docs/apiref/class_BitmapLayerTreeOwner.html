BitmapLayerTreeOwner

The BitmapLayerTreeOwner class is a class that holds a layer tree and can acquire the drawing result as a Bitmap class.
By passing it to the first argument of the Layer class constructor instead of window, the drawing results of those Layers can be acquired as images as a Bitmap class.
By transferring the Bitmap of the drawing result to a Texture, the Layer drawing system can be used with Canvas.
