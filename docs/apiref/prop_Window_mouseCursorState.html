Window.mouseCursorState

Function/Meaning
[Windows*] Mouse cursor display state
Type
Property of Window class
Description

Represents the display state of the mouse cursor.
You can also set the value.
If mcsVisible is specified, the mouse cursor becomes visible.
This is the default state.
If mcsTempHidden is specified, the mouse cursor is hidden, but moving the mouse even slightly changes it to mcsVisible and makes it visible.
Calling the Window.hideMouseCursor method results in this state.
If mcsHidden is specified, the mouse cursor is hidden.
It will not become visible even if the mouse is moved.
On Android, it is always mcsHidden.
