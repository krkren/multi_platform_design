Layer.imeMode

Function/Meaning
IME Mode
Type
Property of Layer class
Description

Represents the IME mode. You can also set the value.
When the layer gains focus, the IME switches to the mode specified here.
The possible values are as follows:

  • If imDisable is specified, the IME is disabled. Input using the IME is not possible, and the user cannot enable the IME manually.
  • If imClose is specified, the IME is disabled. Unlike imDisable, the user can enable the IME manually.
  • If imOpen is specified, the IME is enabled.
  • If imDontCare is specified, the IME's enabled/disabled state is inherited from the previous state. The user can enable or disable the IME manually. In Japanese input, this is the general mode for allowing users to freely input half-width/full-width characters.
  • If imSAlpha is specified, the IME is enabled and enters half-width alphanumeric input mode.
  • If imAlpha is specified, the IME is enabled and enters full-width alphanumeric input mode.
  • If imHira is specified, the IME is enabled and enters Hiragana input mode.
  • If imSKata is specified, the IME is enabled and enters half-width Katakana input mode.
  • If imKata is specified, the IME is enabled and enters full-width Katakana input mode.
  • If imChinese is specified, the IME is enabled and enters a mode that accepts 2-byte Chinese input. This cannot be used in a Japanese environment.
  • If imSHanguel is specified, the IME is enabled and enters a mode that accepts 1-byte Korean input. This cannot be used in a Japanese environment.
  • If imHanguel is specified, the IME is enabled and enters a mode that accepts 2-byte Korean input. This cannot be used in a Japanese environment.

The default value is imDisable.
