Window.setMaskRegion

Function/Meaning

[Windows+] Set window region according to mask

Type
Method of Window class
Syntax
setMaskRegion(threshold=1)
Arguments
threshold

Specifies the mask threshold value.
The window will be clipped to the shape of the parts of the primary layer mask (layer opacity information) that are greater than this value.

Return Value
None (void)
Description

Sets the window region according to the primary layer mask (layer opacity information).
This allows the window to have an irregular shape.
To ensure that the displayed primary layer does not shift from the window size and position, you must do the following:
Set Window.borderStyle to bsNone.
Set Layer.imageLeft and Layer.imageTop to 0.

See Also
