Layer.onTouchUp

Function/Meaning

Finger lifted from screen

Type
Event of the Layer class
Syntax
onTouchUp(x, y, cx, cy, id)
Arguments
x

The x-coordinate (in client coordinates) of the position where the finger was lifted.

y

The y-coordinate (in client coordinates) of the position where the finger was lifted.

cx

The horizontal number of pixels the finger is in contact with.
Always 1 if the device does not support this.

cy

The vertical number of pixels the finger is in contact with.
Always 1 if the device does not support this.

id

Touch ID.
During multi-touch, a unique value is assigned to each position, allowing identification of the position by this ID.

Return Value
None (void)
Description

Occurs when a finger is lifted from the touch panel.
This event occurs when the X and Y coordinates are within this layer.
However, unlike a mouse, it is determined by the current coordinates, so it may not be the same layer where onTouchDown occurred.
