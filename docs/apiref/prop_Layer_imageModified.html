Layer.imageModified

Function/Meaning
Whether the image has been modified
Type
Property of the Layer class
Description

Indicates whether the layer's image has been modified.
You can also set this value.
It is automatically set to true when drawing on the layer's image or changing the image size.
By setting this property to false, it will become true when the layer's image is modified, allowing you to detect changes.
This property itself does not affect the layer's behavior.
