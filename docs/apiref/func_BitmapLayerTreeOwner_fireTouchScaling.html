BitmapLayerTreeOwner.fireTouchScaling

Function/Meaning

Notifies the layer that a scaling operation has been performed

Type
BitmapLayerTreeOwner class method
Syntax
fireTouchScaling(startdist, curdist, cx, cy, flag)
Arguments
startdist

Pixel distance when multi-touch started

curdist

Pixel distance of the touch when the event occurred

cx

x-coordinate of the center position (client coordinates)

cy

y-coordinate of the center position (client coordinates)

flag

Multi-touch state flag.

Return Value
None (void)
Description
