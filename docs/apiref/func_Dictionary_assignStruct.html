Dictionary.assignStruct

Function/Meaning

Copies a dictionary

Type
Method of the Dictionary class
Syntax
assignStruct(src:Dictionary)
Arguments
src

Source dictionary

Return Value
None (void)
Description

Copies the contents of the dictionary specified by the argument.
Unlike the assign method, if a member is an array or a dictionary, its contents are also copied recursively (in the case of the assign method, only the reference is copied).
