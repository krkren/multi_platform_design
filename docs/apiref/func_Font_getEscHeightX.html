Font.getEscHeightX

Function/Meaning

X-coordinate displacement in the vertical direction of characters

Type
Method of the Font class
Syntax
getEscHeightX(text)
Arguments
text

Specifies the string for which you want to obtain the X-coordinate displacement in the vertical direction.

Return Value

Returns the X-coordinate displacement in the vertical direction of characters in pixels.

Description

Obtains the displacement of the character drawing position required when drawing the specified characters with the current font settings.
This method allows you to obtain the displacement on the X-axis when moving vertically (downward) relative to the character drawing direction.
