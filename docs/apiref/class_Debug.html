Debug

The Debug class provides functions related to debugging in KiriKiri.
You cannot create objects from this class.
The name of the KiriKiri console log is krkr.console.log.
Also, the file created when a hardware exception occurs is hwexcept.log.
These log files are in the project directory by default.
However, if the project directory is an archive or a directory that cannot be written to, it will not be output.
The output destination of the log file can be changed with the logLocation property (in the case of KAG, it is set to the save destination of the bookmark data).
On Windows, the console is the command prompt, and on Android, it is output to logCat.
