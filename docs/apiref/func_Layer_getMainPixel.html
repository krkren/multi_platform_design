Layer.getMainPixel

Function/Meaning

Get the color of the main image

Type
Method of the Layer class
Syntax
getMainPixel(x, y)
Arguments
x

Specifies the x-coordinate (in layer image coordinates) from which to get the color.

y

Specifies the y-coordinate (in layer image coordinates) from which to get the color.

Return Value

Color number in 0xRRGGBB format

Description

Gets the color at an arbitrary position in the layer's main image (the image holding the color).
An exception will occur if a position that is invalid (out of range) for the image coordinates is specified.
