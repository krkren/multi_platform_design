System.inform

Function/Meaning

Display a message

Type
Method of the System class
Syntax
inform(text, caption=)
Arguments
text

Specifies the message to display.

caption

Specifies the string to be used as the window caption.

Return Value
None (void)
Description

Displays a window to show a message to the user.
The window is displayed modally (meaning other windows cannot be operated while it is displayed).
On Android, it is displayed as a toast, and control returns immediately.
Also, the caption has no effect.
