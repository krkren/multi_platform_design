Bitmap.Bitmap

Function/Meaning

Constructing a Bitmap object with a specified size

Type
Method of the Bitmap class
Syntax
Bitmap(width, height, bpp=32)
Arguments
width

Specifies the width of the image.

height

Specifies the height of the image.

bpp

Specifies the number of bits per pixel for the image. Specify either 32 or 8.

Return Value
None (void)
Description

Constructs a Bitmap class object with the specified size.
