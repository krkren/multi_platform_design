Clipboard.asText

Function/Meaning
[Windows*] Text format
Type
Property of the Clipboard class
Description

Represents the text format data currently in the clipboard.
Values can also be written.
Reading the value allows you to obtain the text format data currently in the clipboard.
If there is no text format data in the clipboard, void is returned.
Writing a value allows you to store text format data in the clipboard.
Nothing happens on Android. Reading will fail.
