String.[]

Function/Meaning
'numeric' property
Type
Property of String class
Description

By specifying a number as a property indirectly, you can access the character at that position.
For example, if str is a string or an expression representing a string, str[0] will be a string representing the first character.
An exception occurs if the number is beyond the length of the string or is negative (however, an empty string is returned for the position at the end of the string).

Assignment expressions cannot be created.
