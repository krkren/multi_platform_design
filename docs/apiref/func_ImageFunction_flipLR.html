ImageFunction.flipLR

Function/Meaning

Horizontal Flip

Type
Method of ImageFunction class
Syntax
flipLR(bmp, rect=null)
Arguments
bmp

Specifies the Bitmap object to flip.

rect

Specifies the region to flip as a Rect object (in Bitmap image coordinates).
If omitted, the entire image is targeted.

Return Value
None (void)
Description

Performs a horizontal flip of the image.
