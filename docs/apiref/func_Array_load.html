Array.load

Function/Meaning

Loads an array from a file.

Type
Method of the Array class
Syntax
load(storage:string, mode:string=null) :Array
Arguments
storage

The name of the file to load

mode

Specifies the mode string when loading the file.
If "o" is followed by a decimal offset, loading will start from that byte position in the file.

Return Value

Returns this Array class object itself.

Description

Loads an array from a text file.
Each line of text in the file becomes an element in the array, and the count property will eventually equal the number of lines in the file.
