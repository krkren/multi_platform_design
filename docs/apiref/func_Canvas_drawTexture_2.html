Canvas.drawTexture

Function/Meaning

Draws a single texture

Type
Method of the Canvas class
Syntax
drawTexture(texture:Texture)
Arguments
texture

Specifies the texture to be used for drawing

Return Value
None (void)
Description

Drawn using defaultShader.
Position, scale, and rotation are specified via matrix.
In addition to the Texture class, an Offscreen class can also be specified for the texture.
When specifying an Offscreen class, it is assumed that the Offscreen class has been removed from the renderTarget (to avoid circular references).
