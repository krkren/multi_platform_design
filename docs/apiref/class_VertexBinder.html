VertexBinder

VertexBinder is a class for associating VertexBuffer with vertex information to be input to the shader.
Set an instance of this class to the property of ShaderProgram.
The VertexBinder set in ShaderProgram is passed to the shader as vertex information when called by Canvs.drawMesh.
