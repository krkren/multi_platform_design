Layer.onTouchRotate

Function/Meaning

Rotation operation performed

Type
Layer class event
Syntax
onTouchRotate(startangle, currentangle, distance, cx, cy, flag)
Arguments
startangle

The angle in radians when multi-touch started.

currentangle

The angle in radians of the touch when the event occurred.

distance

The pixel distance of the touch when the event occurred.

cx

The x-coordinate of the center position (in client coordinates).

cy

The y-coordinate of the center position (in client coordinates).

flag

Multi-touch state flag.

  • 0x01 : Set for the first event when multi-touch starts.
Return Value
None (void)
Description

Occurs when a rotation operation is performed via multi-touch on a touch panel.
This event occurs on the layer that has focus.
