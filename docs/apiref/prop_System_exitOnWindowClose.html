System.exitOnWindowClose

Function/Meaning
[Windows] Whether to exit when the main window is closed
Type
System class property
Description

Represents whether to exit when the main window (the first instance of the Window class created) is closed.
The value can also be set.
Default is true.
Note that if the main window is closed and no other debug-related windows are displayed, KiriKiri may remain in the system without exiting and become uncontrollable (it may be necessary to terminate the process from the Task Manager).
