Array.save

Function/Meaning

Saves the array to a file.

Type
Method of the Array class
Syntax
save(storage:string, mode:string=null) :Array
Arguments
storage

The name of the file to save to.

mode

Specifies the mode string for writing the file.

  • If "c" is specified, simple encryption is applied to the file (it can still be read directly with the load method).
  • If "z" is specified, the file is compressed and saved (cannot be used with "c"; it can still be read directly with the load method).
  • If "o" is followed by a decimal offset, writing starts from that byte position in the file.
  • If "b" is specified, it is saved in binary format (cannot be used with other mode specifications; it can still be read directly with the load method).
Return Value

Returns the Array class object itself.

Description

Saves the array to a file.
Each member is output line by line, but references to objects, classes, functions, etc., will result in empty lines.
