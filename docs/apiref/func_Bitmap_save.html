Bitmap.save

Function/Meaning

Save image

Type
Bitmap class method
Syntax
save(name, type=bmp, meta=null)
Arguments
name

Specifies the storage name to save to.

type

Specifies the image format to save as a string.
The following formats can be specified in the current version.
"bmp" or "bmp32": 32bpp BMP. Also saves the alpha channel (mask).
"bmp24": 24bpp BMP. Alpha channel (mask) is not saved.
"bmp8": 8bpp BMP. Alpha channel (mask) is not saved. The image is reduced using 4x4 ordered dithering with a fixed 252-color palette.
"jpg": JPEG. Alpha channel (mask) is not saved. The image is saved with 90% quality.
"jpg###": JPEG. Alpha channel (mask) is not saved. The image is saved with the quality specified by ###. "jpg010" is 10%, "jpg100" is 100%, "jpg080" is 80%.
"png": 32bpp PNG. Also saves the alpha channel (mask).
"png24": 24bpp PNG. Alpha channel (mask) is not saved.
"tlg5": 32bpp TLG5. Also saves the alpha channel (mask).
"tlg524": 24bpp TLG5. Alpha channel (mask) is not saved.
"tlg6": 32bpp TLG6. Also saves the alpha channel (mask).
"tlg624": 24bpp TLG6. Alpha channel (mask) is not saved.

meta

For TLG format, specifies tag information to save in dictionary format. For other formats, specifies additional options (since 1.3.0). Available options can be retrieved with getSaveOption.

Return Value
None (void)
Description

Saves the file to the storage (file) specified by name, in the image format specified by type.

Reference
Bitmap.getSaveOption
