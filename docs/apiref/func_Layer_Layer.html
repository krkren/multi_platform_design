Layer.Layer

Function/Meaning

Layer object construction

Type
Layer class method
Syntax
Layer(window, parent)
Arguments
window

Specifies the window (Window class object) that will own this layer.
Once the window is determined, it cannot be changed.

parent

Specifies the layer that will be the parent of this layer. If null is specified, it becomes the primary layer.
The primary layer is the parent of all layers; only one can exist per window, and one must exist if layers are used.
However, depending on the drawing device (settable via Window.drawDevice), some windows may have multiple primary layers.
The parent of a layer can be changed using the Layer.parent property.

Return Value
None (void)
Description

Constructs a Layer class object. The Layer class is constructed in a hidden state.
