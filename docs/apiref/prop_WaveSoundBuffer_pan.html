WaveSoundBuffer.pan

Function/Meaning
Pan
Type
WaveSoundBuffer class property
Description

Represents the pan (sound image position).
The value can also be set.
Specifies the left/right position where the sound is heard.
Specified as a value from -100000 to 0 to 100000, where -100000 is completely left, 0 is center, and 100000 is completely right.
