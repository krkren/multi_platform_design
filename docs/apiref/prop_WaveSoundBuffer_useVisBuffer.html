WaveSoundBuffer.useVisBuffer

Function/Meaning
Whether to use the visualization buffer
Type
Property of the WaveSoundBuffer class
Description

Indicates whether to use the visualization buffer.
This value can also be set.
If set to true, the visualization buffer becomes available, and the WaveSoundBuffer.getVisBuffer method can be used.
The default is false.
Note that setting this to true consumes more memory and CPU time than when it is set to false.
