Dictionary.contains

Function/Meaning

Determines if the specified key exists in the dictionary.

Type
Dictionary class method
Syntax
contains(target:Dictionary, key:string) :bool
Arguments
target

The dictionary to check for existence.

key

The key to check for existence.

Return Value

Whether the key exists.

Description

Returns true if the specified key exists in the dictionary, and false if it does not.
If a context is available, target can be omitted.
This can also be checked using the 'in' operator.
