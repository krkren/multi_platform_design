BasicDrawDevice.onDisplayRotate

Function/Meaning

[Windows+] When the screen is rotated

Type
Event of BasicDrawDevice class
Syntax
onDisplayRotate(orientation, angle, bpp, width, height)
Arguments
orientation

The screen orientation.
It will be one of the following values.

  • oriUnknown (Acquisition failed/Unknown)
  • oriPortrait (Portrait)
  • oriLandscape (Landscape)
angle

Angle.
The angle is one of 0, 90, 180, 270, -1, and -1 if it could not be acquired.
The angle is the rotation angle from the device default, so it is 0 in portrait orientation on portrait devices.
For normal devices, 0 is often landscape.
Portrait orientation is 0 on recent 8-inch tablets, etc., where portrait orientation is the standard orientation.

bpp

bits per pixel.

width

Screen width.

height

Screen height.

Return Value
None (void)
Description

Represents the event function that is called when the screen is rotated.
