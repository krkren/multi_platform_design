onDisplayRotate - [Windows+] When the screen is rotated Event of BasicDrawDevice class Function/Meaning Represents the event function that is called when the screen is rotated. Type Event of BasicDrawDevice class Syntax onDisplayRotate(orientation, angle, bpp, width, height) Arguments orientation: Screen orientation oriUnknown (Acquisition failed/Unknown) oriPortrait (Portrait) oriLandscape (Landscape) angle: Rotation angle The angle is one of 0, 90, 180, 270, or -1. It is -1 if it could not be acquired. The angle is the rotation angle from the device default, so it is 0 in portrait orientation on portrait-first devices. For typical devices, 0 is often landscape. On recent 8-inch tablets where portrait is the standard orientation, portrait will be 0. bpp: Bits per pixel. width: Screen width. height: Screen height. Return Value None Description Represents the event function that is called when the screen is rotated.
