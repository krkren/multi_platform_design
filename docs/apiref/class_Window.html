Window

Window class is a class for managing windows.
Window size control is only possible on Windows.
On Android, it depends on the screen size.
Generate an Offscreen class with the size you want to render, and draw normally there.
Scaling etc. according to the window (screen) size is controlled by a script.
It is also possible to display UI, characters, etc. neatly at the same size using a script.

Events that Android does not support will not occur on Android.
