BitmapLayerTreeOwner.fireMouseWheel

Function/Meaning

Notifies the layer of mouse wheel rotation

Type
BitmapLayerTreeOwner class method
Syntax
fireMouseWheel(shift, delta, x, y)
Arguments
shift

The state of shift keys and mouse buttons pressed simultaneously when the mouse moved

delta

The rotation angle of the wheel

x

The x-coordinate where the wheel was rotated (client coordinates)

y

The y-coordinate where the wheel was rotated (client coordinates)

Return Value
None (void)
Description
