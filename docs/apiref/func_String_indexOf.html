String.indexOf

Function/Meaning

Returns the position of a substring

Type
Method of the String class
Syntax
indexOf(text:string, offset:int=void) :int
Arguments
text

Substring

offset

Search start position

Return Value

Returns 0 if it is at the beginning of the string, and -1 if it was not found.

Description

Searches for text within the string starting from offset and returns the position where it was first found.
If offset is omitted, the search starts from the beginning of the string.
