System.graphicCacheLimit

Function/Meaning
Image cache limit
Type
Property of the System class
Description

Represents the KiriKiri image cache limit in bytes.
The value can also be set.
If gcsAuto is specified, the value is automatically set according to the amount of memory installed in the machine.
Rule images and region images consume bytes equal to width x height.
Other images consume bytes equal to width x height x 4.
