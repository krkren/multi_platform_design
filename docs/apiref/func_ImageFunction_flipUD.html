ImageFunction.flipUD

Function/Meaning

Vertical Flip

Type
Method of ImageFunction class
Syntax
flipUD(bmp, rect=null)
Arguments
bmp

Specifies the Bitmap object to flip.

rect

Specifies the region to flip as a Rect object (relative to the Bitmap's image position).
If unspecified, the entire area is targeted.

Return Value
None (void)
Description

Performs a vertical flip of the image.
