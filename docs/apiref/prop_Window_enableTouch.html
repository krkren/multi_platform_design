Window.enableTouch

Function/Meaning
Enable/Disable Touch Events
Type
Property of the Window class
Description

Indicates whether touch events are enabled.
This value can also be set.
If set to true, events such as Window.onTouchDown will be enabled, and touch operations will no longer trigger Window.onMouseDown, etc.
In environments with a touch device and multi-touch enabled, the default is true.
On Android, touch is always enabled.
