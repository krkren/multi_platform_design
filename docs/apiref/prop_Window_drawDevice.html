Window.drawDevice

Function/Meaning
[Windows+] Drawing device
Type
Property of Window class
Description

Represents the drawing device object.
Drawing devices are now deprecated.
Drawing using Canvas is recommended.
A value can also be set.
When a value is set, the drawing device previously assigned to this window is automatically invalidated.
By default, an instance of the Window.BasicDrawDevice class is assigned.
For details on Window.BasicDrawDevice, please also refer to the description in core/visual/win32/BasicDrawDevice.cpp of the Kirikiri source code.
When specifying a custom drawing device (provided by a plugin), please follow that plugin's documentation.
