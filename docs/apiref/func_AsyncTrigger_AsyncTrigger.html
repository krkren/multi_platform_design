AsyncTrigger.AsyncTrigger

Function/Meaning

Construction of AsyncTrigger object

Type
Method of the AsyncTrigger class
Syntax
AsyncTrigger(owner, actionname=action)
Arguments
owner

Specifies the target where the event will occur.

actionname

Specifies the name of the method to receive the event in the target object specified by owner.
If an empty string is specified, owner is treated as a method, and owner is called directly every time an event occurs.

Return Value
None (void)
Description

Constructs an object of the AsyncTrigger class.
