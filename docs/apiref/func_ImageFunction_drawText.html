ImageFunction.drawText

Function/Meaning

Text Drawing

Type
Method of ImageFunction class
Syntax
drawText(bmp, font, x, y, text, color, opa=255, aa=true, face=dfAlpha, shadowlevel=0, shadowcolor=0, shadowwidth=0, shadowofsx=0, shadowofsy=0, hda=false, cliprect=null)
Arguments
bmp

Specifies the Bitmap object to draw text on.

font

Specifies the font to draw with as a Font object.

x

Specifies the x-coordinate (in image coordinates) of the origin to start drawing text, in pixels.

y

Specifies the y-coordinate (in image coordinates) of the origin to start drawing text, in pixels.

text

Specifies the text to draw.

color

Specifies the color of the text to draw in 0xRRGGBB format.

opa

Specifies the opacity of the text to draw (-255 to 0 to 255).
Specifying a negative number is only valid when face is dfAlpha; in this case, opacity is removed in the shape of the characters (the smaller the value, the greater the effect).

aa

Specifies whether to perform anti-aliasing.
If true is specified, anti-aliasing is performed.
If false is specified, it is not performed.

face

Specifies the drawing method.

  • If dfAlpha is specified, the image is treated as an image with an alpha channel and drawn.
  • If dfAddAlpha is specified, the image is treated as an image with an additive alpha channel and drawn.
  • If dfOpaque is specified, the image is treated as completely opaque and drawn.
shadowlevel

Specifies the opacity of the shadow.
The appropriate value varies depending on the shadowwidth argument.
If 0 is specified, no shadow is drawn.

shadowcolor

Specifies the shadow color in 0xRRGGBB format.

shadowwidth

Specifies the shadow width (blur).
0 is the sharpest (no blur), and larger values blur the shadow.

shadowofsx

Specifies the x-coordinate of the shadow position in pixels.
If 0 is specified, the shadow is drawn directly below.

shadowofsy

Specifies the y-coordinate of the shadow position in pixels.
If 0 is specified, the shadow is drawn directly below.

hda

Specifies whether to protect the alpha channel.

cliprect

Specifies the clipping region as a Rect object (in Bitmap image coordinates).
If unspecified, the entire area is targeted.

Return Value

Drawing area (Rect object)

Description

Draws text on the Bitmap.
When dfOpaque is specified for face, whether the destination mask is destroyed or protected depends on hda.
