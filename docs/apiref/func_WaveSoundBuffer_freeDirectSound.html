WaveSoundBuffer.freeDirectSound

Function/Meaning

Release DirectSound

Type
Method of the WaveSoundBuffer class
Syntax
freeDirectSound()
Arguments
Return Value
None (void)
Description

Releases DirectSound.
All WaveSoundBuffer class objects will enter a stopped state.
Use this method to release DirectSound in environments where DirectSound and WaveMapper (MCI, etc.) playback cannot occur simultaneously.
This method exists only on the WaveSoundBuffer class itself (it is not available on objects created from WaveSoundBuffer).
To use it, call WaveSoundBuffer.freeDirectSound();.
