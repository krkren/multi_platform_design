ImageFunction.operateStretch

Function/Meaning

Scaled Arithmetic Composition

Type
ImageFunction class method
Syntax
operateStretch(dst, src, dstrect=null, srcrect=null, cliprect=null, mode=omAuto, face=dfAlpha, opa=255, type=stNearest, hda=false, option=1)
Arguments
dst

Specifies the destination Bitmap object.

src

Specifies the source Bitmap object.

dstrect

Specifies the destination rectangle as a Rect object in pixels (relative to the destination Bitmap's image position).
If unspecified, the entire area is targeted.

srcrect

Specifies the source rectangle to be composed as a Rect object in pixels (relative to the source Bitmap's image position).
If unspecified, the entire area is targeted.

cliprect

Specifies the clipping rectangle as a Rect object (relative to the destination Bitmap's image position).
If unspecified, no clipping is performed.

mode

Specifies the arithmetic mode.

  • If omPsNormal is specified, Photoshop-compatible alpha blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsAdditive is specified, Photoshop-compatible Linear Dodge blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsSubtractive is specified, Photoshop-compatible Linear Burn blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsMultiplicative is specified, Photoshop-compatible Multiply blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsScreen is specified, Photoshop-compatible Screen blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsOverlay is specified, Photoshop-compatible Overlay blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsHardLight is specified, Photoshop-compatible Hard Light blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsSoftLight is specified, Photoshop-compatible Soft Light blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsColorDodge is specified, Photoshop-compatible Color Dodge blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsColorDodge5 is specified, Color Dodge blending compatible with Photoshop version 5.x or lower is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsColorBurn is specified, Photoshop-compatible Color Burn blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsLighten is specified, Photoshop-compatible Lighten blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsDarken is specified, Photoshop-compatible Darken blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsDifference is specified, Photoshop-compatible Difference blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsDifference5 is specified, Difference blending compatible with Photoshop version 5.x or lower is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omPsExclusion is specified, Photoshop-compatible Exclusion blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omAdditive is specified, additive blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omSubtractive is specified, subtractive blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omMultiplicative is specified, multiplicative blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omDodge is specified, dodge blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omDarken is specified, darken blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omLighten is specified, lighten blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omScreen is specified, screen multiply blending is performed (implemented for types other than stNearest and stFastLinear in version 1.3 and later).
  • If omAlpha is specified, alpha blending is performed. If omAddAlpha is specified, additive alpha blending is performed. In this case, if face is dfOpaque and hda is false, linear interpolation is possible by specifying stFastLinear for the type argument.
  • If omOpaque is specified, the alpha information of src is ignored, and src is always considered completely opaque. In this case, if face is dfOpaque and hda is false, linear interpolation is possible by specifying stFastLinear for the type argument.
face

Specifies the drawing method.

  • If dfAlpha is specified, the image is treated as an image with an alpha channel and drawn.
  • If dfAddAlpha is specified, the image is treated as an image with an additive alpha channel and drawn.
  • If dfOpaque is specified, the image is treated as completely opaque and drawn.
opa

Specifies the intensity of the operation (0 to 255).

type

Specifies the scaling type.

  • stNearest : Nearest neighbor method is used.
  • stFastLinear : Low-precision linear interpolation is used (partially implemented).
  • stSemiFastLinear : Fixed-point linear interpolation is used (version 1.3 and later).
  • stLinear : Linear interpolation is used (implementation changed in version 1.3 and later).
  • stFastCubic : Fixed-point cubic interpolation is used (version 1.3 and later).
  • stCubic : Cubic interpolation is used (implementation changed in version 1.3 and later).
  • stFastLanczos2 : Fixed-point Lanczos interpolation (4x4 range) is used (version 1.3 and later).
  • stLanczos2 : Lanczos interpolation (4x4 range) is used (version 1.3 and later).
  • stFastLanczos3 : Fixed-point Lanczos interpolation (6x6 range) is used (version 1.3 and later).
  • stLanczos3 : Lanczos interpolation (6x6 range) is used (version 1.3 and later).
  • stFastSpline16 : Fixed-point spline interpolation (4x4) is used (version 1.3 and later).
  • stSpline16 : Spline interpolation (4x4) is used (version 1.3 and later).
  • stFastSpline36 : Fixed-point spline interpolation (6x6) is used (version 1.3 and later).
  • stSpline36 : Spline interpolation (6x6) is used (version 1.3 and later).
  • stFastAreaAvg : Fixed-point area average reduction is used. Scaling up is not possible (version 1.3 and later).
  • stAreaAvg : Area average reduction is used. Scaling up is not possible (version 1.3 and later).
  • stFastGaussian : Fixed-point Gaussian interpolation (4x4) is used (version 1.3 and later).
  • stGaussian : Gaussian interpolation (4x4) is used (version 1.3 and later).
  • stFastBlackmanSinc : Fixed-point Blackman-Sinc interpolation (8x8) is used (version 1.3 and later).
  • stBlackmanSinc : Blackman-Sinc interpolation (8x8) is used (version 1.3 and later).
