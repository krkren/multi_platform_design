Font.mapPrerenderedFont

Function/Meaning

Mapping of pre-rendered fonts

Type
Font class method
Syntax
mapPrerenderedFont(fontstorage)
Arguments
fontstorage

Specifies the pre-rendered font storage.

Return Value
None (void)
Description

Maps a pre-rendered font to the currently selected font name.
From then on, the specified pre-rendered font will be used instead for fonts with the same settings.
The settings will be effective for all layers.

See Also
Font.unmapPrerenderedFont
