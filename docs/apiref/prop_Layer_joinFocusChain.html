Layer.joinFocusChain

Function/Meaning
Whether to join the focus chain
Type
Layer class property
Description

Represents whether to join the focus chain.
If set to true, the layer joins the focus chain, appearing in Layer.prevFocusable and allowing focus to move to the layer via the TAB key, etc.
If set to false, it does not join the focus chain, but it can still receive focus via the Layer.focus method, etc.
