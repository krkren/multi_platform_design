Font.getEscWidthY

Function/Meaning

Y-coordinate displacement in the horizontal direction of characters

Type
Method of Font class
Syntax
getEscWidthY(text)
Arguments
text

Specifies the string for which you want to obtain the Y-coordinate displacement in the horizontal direction.

Return Value

Returns the Y-coordinate displacement in the horizontal direction of characters in pixels.

Description

Obtains the displacement of the character drawing position required when drawing the specified characters with the current font settings.
This method allows you to obtain the displacement on the Y-coordinate when moving horizontally relative to the character drawing direction.
