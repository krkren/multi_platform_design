System.personalPath

Function/Meaning
My Documents path
Type
Property of the System class
Description

Represents the path to the user's My Documents folder.
In Windows, the folder represented by "Personal" in the registry key HKEYCURRENTUSER\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders is returned.
Usually this points to the "My Documents" folder.
If this folder does not exist, it returns the same folder as System.exePath.
