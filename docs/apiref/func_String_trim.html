String.trim

Function/Meaning

Returns a string with whitespace removed from the beginning and end of the string

Type
Method of the String class
Syntax
trim() :string
Arguments
Return Value

Returns a string with leading and trailing whitespace characters removed.

Description

Does not affect the original string.
Leading and trailing whitespace refers to characters with character codes U+0001 to +0020.
In other words, this includes ASCII control characters, but does not include so-called full-width spaces.
If there are consecutive spaces at the beginning or end, they will all be removed.
