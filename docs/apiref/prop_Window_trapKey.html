Window.trapKey

Function/Meaning
[Windows*] Whether to trap key input
Type
Property of the Window class
Description

Indicates whether to trap key input.
You can also set the value.
If set to true, key input events occurring in other windows can be intercepted and triggered on this window.
If this property is set to true for multiple windows, the key input event will occur for the window created most recently.
On Android, this is always false.
