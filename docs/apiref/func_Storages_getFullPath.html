Storages.getFullPath

Function/Meaning

Get full unified storage name

Type
Storages class method
Syntax
getFullPath(path)
Arguments
path

Specifies the storage name to be converted into a full unified storage name.

Return Value

Full unified storage name

Description

Converts the storage name specified by path into a full unified storage name.
Redundant path access (e.g., system/flags/../data/) is fully compressed.
If the current media or current folder is not specified, it will be complemented.
