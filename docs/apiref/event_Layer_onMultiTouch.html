Layer.onMultiTouch

Function/Meaning

Multi-touch state changed

Type
Event of the Layer class
Syntax
onMultiTouch()
Arguments
Return Value
None (void)
Description

Occurs when a multi-touch state starts, moves, or ends.
Coordinate information can be obtained using the Window.touchPointCount property and the Window.getTouchPoint method.
This event occurs on the layer that has focus.

Reference
Window.getTouchPoint
Window.touchPointCount
