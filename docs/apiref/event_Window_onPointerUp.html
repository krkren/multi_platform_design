Window.onPointerUp

Function/Meaning

A pointing device was released

Type
Event of the Window class
Syntax
onPointerUp(type, x, y, cx, cy, shift, id)
Arguments
type

Type of touch or mouse button

x

X position

y

Y position

cx

Contact width (pixels), 1 for mouse

cy

Contact height (pixels), 1 for mouse

shift

Modifier key state, 0 for touch

id

Touch ID, always 0 for mouse

Return Value
None (void)
