Layer.onSearchNextFocusable

Function/Meaning

Search for the next focusable layer

Type
Layer class event
Syntax
onSearchNextFocusable(layer)
Arguments
layer

The layer object that receives focus.
Specify this as an argument when calling the superclass method of the same name.
If null is passed, it is processed as if there are no focusable layers.

Return Value
None (void)
Description

This event occurs when searching for the next focusable layer.
To specify the layer that receives focus, pass that layer as an argument to the superclass event of the same name.
