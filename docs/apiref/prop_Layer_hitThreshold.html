Layer.hitThreshold

Function/Meaning
Hit detection threshold
Type
Layer class property
Description

Represents the threshold for mouse event hit detection.
The value can also be set.
This property is only valid when the Layer.hitType property is htMask. Mouse messages are received if the mask (opacity) image value is greater than or equal to the value specified by this property.
If 0 is specified, all mouse messages are received.
If 256 is specified, no mouse messages are received.
The default value is 16.
