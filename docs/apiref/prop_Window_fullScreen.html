Window.fullScreen

Function/Meaning
[Windows*] Whether it is full screen
Type
Property of the Window class
Description

Represents whether it is full screen. You can also set the value.
If true is specified, the client area of the current window becomes full screen.
The screen resolution during full screen will be the size of the client area.
If false is specified, it will be displayed in a window.
On Android, it is always fixed to full screen.
