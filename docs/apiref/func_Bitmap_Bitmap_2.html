Bitmap.Bitmap

Function/Meaning

Constructing a Bitmap object by specifying a file

Type
Method of the Bitmap class
Syntax
Bitmap(filename, colorkey=clNone)
Arguments
filename

Specifies the image file name.

colorkey

Specifies the color key. The value is the same as Layer.loadImages.

Return Value
None (void)
Description

Constructs a Bitmap class object from a file.
