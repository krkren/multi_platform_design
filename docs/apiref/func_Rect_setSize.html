Rect.setSize

Function/Meaning

Set rectangle size

Type
Rect class method
Syntax
setSize(width, height)
Arguments
width

Specifies the width of the rectangle.
This value can also be retrieved or set via the Rect.width property.

height

Specifies the height of the rectangle.
This value can also be retrieved or set via the Rect.height property.

Return Value
None (void)
Description

Specifies the rectangle size.
