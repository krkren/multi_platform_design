VideoOverlay.selectAudioStream

Function/Meaning

[Windows*] Select audio stream

Type
Method of the VideoOverlay class
Syntax
selectAudioStream(streamNumber)
Arguments
streamNumber

Specifies the audio stream number.

Return Value
None (void)
Description

Enables the specified audio stream number.
Cannot be used with videos that do not contain multiple audio streams.
On Android, this always returns 0 and setting it has no effect.
