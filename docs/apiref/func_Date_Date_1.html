Date.Date

Function/Meaning

Date string constructor

Type
Method of the Date class
Syntax
Date(date:string)
Arguments
date

Date string

Return Value
None (void)
Description

The date string must be specified in one of the following formats: ANSI C asctime() format, RFC1123 format, RFC1036 format, or "YYYY/MM/DD HH:MM:SS" format.
Recognition of the date string is performed automatically and supports some variations in format, but it may not be recognized if it differs significantly from the expected formats.
