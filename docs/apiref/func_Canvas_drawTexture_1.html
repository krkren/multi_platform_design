Canvas.drawTexture

Function/Meaning

Draw 3 textures using a specified shader

Type
Method of Canvas class
Syntax
drawTexture(texture:Texture, texture2:Texture, texture3:Texture, shader:ShaderProgram)
Arguments
texture

Specifies the texture to be used for drawing

texture2

Specifies the second texture to be used for drawing

texture3

Specifies the third texture to be used for drawing

shader

Specifies the shader to be used for drawing

Return Value
None (void)
Description

In OpenGL ES 2.0, the maximum number of textures is 8. In 3.0, it is up to 16 on the vertex side and 16 on the fragment side, so more can be added, but for now, only up to 3 are defined.
In the future, we are considering creating a version that passes textures as an array to support 4 or more textures.
