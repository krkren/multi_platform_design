RegExp.exec

Function/Meaning

Performs regular expression pattern matching on the specified string and returns an array containing the match results

Type
Method of the RegExp class
Syntax
exec(text:string) :Array
Arguments
text

String

Return Value

If the pattern does not match, the number of elements in the array will be 0.
If a match is found, element 0 (the first element) contains the entire matched portion, and subsequent elements contain each captured sub-match (the parts specified by ( ) in the regular expression pattern).

Description

This method updates the values of each property of this object.
