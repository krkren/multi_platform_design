Texture.Texture

Function/Meaning

File loading constructor

Type
Method of Texture class
Syntax
Texture(filename:string, format:int=tcfRGBA, is9patch:bool=false)
Arguments
filename

Image filename

format

Color format (tcfRGBA or tcfAlpha). When tcfAlpha is selected, color information is converted to grayscale.

is9patch

Whether to load 9patch information and use it for 9patch rendering.

Return Value
None (void)
