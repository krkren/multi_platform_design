RegExp.RegExp

Function/Meaning

Constructor

Type
Method of RegExp class
Syntax
RegExp(pattern:string, flag:string=null)
Arguments
pattern

Pattern string

flag

Flag string

Return Value
None (void)
Description

The pattern string is a regular expression pattern.
Note that when specifying \ in the pattern, it must be written as \\ within a string literal.
The flag string is a combination of the characters g and i.
g is the global search flag.
When this flag is set, the match, exec, and replace methods will process all parts of the string that match the pattern.
The i flag makes the search case-insensitive.
