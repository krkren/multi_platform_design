ImageFunction.drawGlyph

Function/Meaning

Character Drawing

Type
Method of ImageFunction class
Syntax
drawGlyph(bmp, x, y, glyph, color, opa=255, aa=true, face=dfAlpha, shadowlevel=0, shadowcolor=0, shadowwidth=0, shadowofsx=0, shadowofsy=0, hda=false, cliprect=null)
Arguments
bmp

Specifies the Bitmap object to draw the character on.

x

Specifies the x-coordinate (in image coordinates) of the origin where character drawing starts, in pixels.

y

Specifies the y-coordinate (in image coordinates) of the origin where character drawing starts, in pixels.

glyph

Specifies the glyph to draw.

color

Specifies the color of the character to draw in 0xRRGGBB format.

opa

Specifies the opacity of the character to draw (-255 to 0 to 255).
Specifying a negative number is only valid when face is dfAlpha; in this case, opacity is removed in the shape of the character (the smaller the value, the greater the effect).

aa

Specifies whether to perform anti-aliasing.
If true is specified, anti-aliasing is performed.
If false is specified, it is not performed.

face

Specifies the drawing method.

  • If dfAlpha is specified, the image is treated as an image with an alpha channel and drawn.
  • If dfAddAlpha is specified, the image is treated as an image with an additive alpha channel and drawn.
  • If dfOpaque is specified, the image is treated as completely opaque and drawn.
shadowlevel

Specifies the opacity of the shadow.
The appropriate value varies depending on the value of the shadowwidth argument.
If 0 is specified, no shadow is drawn.

shadowcolor

Specifies the color of the shadow in 0xRRGGBB format.

shadowwidth

Specifies the width (blur) of the shadow.
0 is the sharpest (no blur), and increasing the value allows you to blur the shadow.

shadowofsx

Specifies the x-coordinate value of the shadow position in pixels.
If 0 is specified, the shadow is drawn directly below.

shadowofsy

Specifies the y-coordinate value of the shadow position in pixels.
If 0 is specified, the shadow is drawn directly below.

hda

Specifies whether to protect the alpha channel.

cliprect

Specifies the clipping region as a Rect object (in Bitmap image coordinates).
If unspecified, the entire area is targeted.

Return Value

Drawing area (Rect object)

Description

Draws a glyph on the Bitmap.
When dfOpaque is specified for face, whether the destination mask is destroyed or protected depends on hda.
For the glyph, specify an array in a format like glyph : Array[9] = [ width, height, originx, originy, incx, incy, inc, bitmap(Octet), colors ].
If the glyph's colors are omitted, it is assumed to be 256 levels.
