Layer.flipUD

Function/Meaning

Flip vertically

Type
Method of Layer class
Syntax
flipUD()
Arguments
Return Value
None (void)
Description

Flips the image vertically.
This method is not affected by the drawing clip rectangle set by the Layer.setClip method, etc. (the entire layer image is always flipped).
Additionally, it is not affected by the Layer.face property or the Layer.holdAlpha property.
