System.assignMessage

Function/Meaning

Change message assignment

Type
Method of System class
Syntax
assignMessage(id, msg)
Arguments
id

Specifies the message ID to assign.

msg

Specifies the message to assign to the ID specified by id.

Return Value

Returns true if the ID exists and the message assignment was successful, false otherwise.

Description

Changes the message assignment. You can use this method to change the built-in messages of KiriKiri to different messages.
Usually described in the message map file (see "How to Start").
To get a list of configurable IDs and the messages currently assigned to them, execute "Create Message Map File".
