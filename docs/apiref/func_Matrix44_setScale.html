Matrix44.setScale

Function/Meaning

Set scale

Type
Method of Matrix44 class
Syntax
setScale(x:real, y:real, z:real=1)
Arguments
x

X-axis scale factor to set

y

Y-axis scale factor to set

z

Z-axis scale factor to set

Return Value
None (void)
Description

Sets the values for the XYZ scale in the matrix.
This only sets the values and does not perform operations with the current matrix.
