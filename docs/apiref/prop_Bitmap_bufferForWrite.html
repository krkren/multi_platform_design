Bitmap.bufferForWrite

Function/Meaning
Image buffer pointer (for writing)
Type
Property of the Bitmap class
Description

Represents a pointer to the upper-left corner of the image buffer (a 32bpp bitmap containing color and mask (opacity) information).
This property is provided to allow direct access to the image buffer for plugins and other purposes.
It is returned as an integer type, but in plugins, etc., please cast it to an appropriate type (such as unsigned long *).
Unlike Bitmap.buffer, you can write values to the pointer obtained with this property.
Inside Kirikiri, identical images are shared among multiple objects, but accessing this property will break that sharing state.
The size of the image is represented by the Bitmap.width and Bitmap.height properties.
Refer to Bitmap.bufferPitch for how to calculate the pointer.
