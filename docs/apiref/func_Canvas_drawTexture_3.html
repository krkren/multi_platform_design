Canvas.drawTexture

Function/Meaning

Draws a single texture using a specified shader

Type
Method of the Canvas class
Syntax
drawTexture(texture:Texture, shader:ShaderProgram)
Arguments
texture

Specifies the texture to be used for drawing

shader

Specifies the shader to be used for drawing

Return Value
None (void)
Description

Position, scaling, and rotation are specified via the matrix.
In addition to the Texture class, the Offscreen class can also be specified for the texture.
When specifying an Offscreen class, it is assumed that the Offscreen class has been removed from the renderTarget (to avoid circular references).
