Bitmap.copyFrom

Function/Meaning

Copying images

Type
Method of the Bitmap class
Syntax
copyFrom(src)
Arguments
src

Specifies the source Bitmap object.

Return Value
None (void)
Description

Copies the image.
Although it is called a copy, in reality, it simply results in a state where "the same image is shared by two or more objects," so this method takes almost no execution time.
