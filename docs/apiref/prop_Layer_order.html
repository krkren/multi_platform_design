Layer.order

Function/Meaning
Relative position
Type
Property of the Layer class
Description

Represents the order among sibling layers sharing the same parent.
The smaller the value, the further back it is displayed.
Setting this value allows you to change the order among sibling layers. Setting this value will set the parent layer's Layer.absoluteOrderMode property to false.
