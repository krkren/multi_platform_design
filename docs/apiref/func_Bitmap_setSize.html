Bitmap.setSize

Function/Meaning

Sets the image size

Type
Method of Bitmap class
Syntax
setSize(width, height)
Arguments
width

Specifies the width of the image in pixels. This value can also be retrieved or set via the Bitmap.width property.

height

Specifies the height of the image in pixels. This value can also be retrieved or set via the Bitmap.height property.

Return Value
None (void)
Description

Specifies the image size.
