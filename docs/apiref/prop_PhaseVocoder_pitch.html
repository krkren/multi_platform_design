PhaseVocoder.pitch

Function/Meaning
Frequency axis scale
Type
PhaseVocoder class property
Description

Represents the scale along the frequency axis.
This value can also be set.
The value must be greater than 0.0.
Specifying 1.0 results in no change to the frequency axis scale (the pitch remains unchanged).
For example, specifying 1.0594630943593 raises the pitch by one semitone.
For example, specifying 0.9438743126817 lowers the pitch by one semitone.
Even if both the time and pitch properties are set to 1, the Phase Vocoder filter remains active, which may slightly degrade sound quality and consume CPU time.
If both are to remain at 1, it is recommended to remove this filter from the filter array.
Changes to this property are reflected in real-time, but due to buffering, it takes approximately 2 seconds before the change is audible in playback.
