Bitmap.buffer

Function/Meaning
Image buffer pointer
Type
Property of Bitmap class
Description

Represents a pointer to the upper-left corner of the image buffer (a 32bpp bitmap containing color and mask (opacity) information).
This property exists to provide a means of direct access to the image buffer for plugins and other purposes.
It is returned as an integer type, but should be cast to an appropriate type (such as const unsigned long *) in plugins.
Do not write values to the pointer obtained with this property.
You can write to the pointer if it was obtained via Bitmap.bufferForWrite.
The image size is represented by the Bitmap.width and Bitmap.height properties.
Refer to Bitmap.bufferPitch for the pointer calculation method.
