Timer.mode

Function/Meaning
Mode
Type
Timer class property
Description

Represents the operation mode.
You can also set the value.
Specify one of the following values.

  • atmNormal : Events are delivered with normal event delivery priority.
  • atmExclusive : Delivered with priority over other asynchronous events.
  • atmAtIdle : Delivered when in an idle state (when there are no other events to deliver).
