Bitmap.bufferPitch

Function/Meaning
Image buffer pitch
Type
Property of Bitmap class
Description

Represents the pitch (number of bytes to the next scanline) of the image buffer for the image (a 32bpp bitmap containing color and mask (opacity) information).
This property is intended to provide a means of direct access to the image buffer for plugins and other purposes.
Assuming tjsuint32 is a 32-bit integer type and tjsuint8 is an 8-bit (1-byte) integer type, the pointer to the image position (x, y) can be calculated in C as follows:
( (tjsuint32*)( (tjsuint8)buffer + ybufferPitch )) + x
Note that this property returns the number of bytes, not the number of pixels, to the next scanline.
This value may be slightly larger than the number of bytes strictly required for the image width.
Note that this property can have a negative value.
