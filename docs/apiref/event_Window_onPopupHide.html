Window.onPopupHide

Function/Meaning

[Windows*] Close popup window

Type
Window class event
Syntax
onPopupHide()
Arguments
Return Value
None (void)
Description

This event occurs when the popup window should be closed.
This event occurs when the Window.stayOnTop property is true and the Window.focusable property is false, and "another window is clicked" or "another application becomes active".
Normally, you should perform processing to close or hide the window here.
