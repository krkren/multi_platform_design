Window.onKeyDown

Function/Meaning

Key pressed

Type
Window class event
Syntax
onKeyDown(key, shift)
Arguments
key

The virtual key code value of the pressed key.

shift

The state of shift keys and mouse buttons pressed simultaneously with the key.
A combination of the following values using bitwise OR.

  • ssAlt : ALT key was pressed
  • ssShift : SHIFT key was pressed
  • ssCtrl : CTRL key was pressed
  • ssLeft : Left mouse button was pressed
  • ssMiddle : Middle mouse button was pressed
  • ssRight : Right mouse button was pressed

Additionally, if the keyboard is held down and key repeat occurs, the following value is also combined.

  • ssRepeat : Key repeat occurred
Return Value
None (void)
Description

Occurs when a key is pressed.
