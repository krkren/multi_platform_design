Scripts.execStorage

Function/Meaning

Execute script on storage

Type
Scripts class method
Syntax
execStorage(storage, mode=, context)
Arguments
storage

Specifies the storage to execute.

mode

Specifies the mode string when reading the file.
If "o" is followed by a decimal offset, reading starts from that byte position in the file.

context

Specifies the execution context.

Return Value

Returns the result of the script execution.

Description

Reads the specified storage and executes its content as a TJS2 script.
Exceptions occurring during script execution are not caught within this method, so they can be caught by the caller of this method.

Reference
