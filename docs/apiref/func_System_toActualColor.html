System.toActualColor

Function/Meaning

Get actual color from color constant

Type
System class method
Syntax
toActualColor(color)
Arguments
color

Specifies the color constant (refer to the Color Constant List).
If a standard 0xRRGGBB format color is specified, the value is returned as-is.

Return Value

The actual color represented by the specified color constant is returned in 0xRRGGBB format.

Description

Converts a color constant to an actual color and returns it in 0xRRGGBB format.
