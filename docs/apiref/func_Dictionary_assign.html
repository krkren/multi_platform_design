Dictionary.assign

Function/Meaning

Copies a dictionary array

Type
Method of the Dictionary class
Syntax
assign(src:Dictionary, clear:bool=true)
Arguments
src

Source dictionary array

clear

Whether to clear the contents

Return Value
None (void)
Description

Copies the contents of another dictionary array specified by the argument.
If clear is false, the contents of the destination (the object executing the method) are not cleared, and the contents of the source dictionary array are overwritten.
If a member with the same name as the source dictionary array exists in the destination dictionary array, the destination is overwritten with the source's content.
If an array (Array class object) is specified as the source, it is assumed that the array contains elements that should be members of this dictionary array in the order of name and value, and the contents of that array are copied to this dictionary array.
