VideoOverlay.onStatusChanged

Function/Meaning

Status changed

Type
Event of the VideoOverlay class
Syntax
onStatusChanged(status)
Arguments
status

Represents the status string. It is one of the following:

  • "unload" : Media is not open
  • "ready" : Media preparation is complete
  • "play" : Media is playing
  • "stop" : Media is stopped
  • "pause" : Media is paused
  • "idle" : [Android+] Media is idle.
  • "buffering" : [Android+] Media is buffering.
  • "load error" : [Android+] Media loading error.
  • "player error" : [Android+] Media player error.
Return Value
None (void)
Description

Occurs when the status of this object changes.
"ready" only occurs in vomMFEVR mode or on Android.
