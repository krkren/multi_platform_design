VideoOverlay.playRate

Function/Meaning
[Windows*] Playback rate
Type
Property of the VideoOverlay class
Description

Sets the playback rate of the media.
Specifying 1.0 results in normal playback speed, 0.5 results in half speed, and 2 results in double speed.
The range of possible values depends on the DirectShow filter.
Reference: For MPEG files with audio, values from greater than 0.0 up to 2.0 can be set.
For MPEG files without audio, values from greater than 0.0 up to the range of a double (probably) can be set, but the actual playback speed is limited by processing power.
On Android, this always returns 0. Setting it does nothing.
