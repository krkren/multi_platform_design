Layer.onTouchMove

Function/Meaning

Finger moved

Type
Event of Layer class
Syntax
onTouchMove(x, y, cx, cy, id)
Arguments
x

The x-coordinate of the touch position (in client coordinates).

y

The y-coordinate of the touch position (in client coordinates).

cx

The number of horizontal pixels the finger is in contact with.
Always 1 if the device does not support this.

cy

The number of vertical pixels the finger is in contact with.
Always 1 if the device does not support this.

id

Touch ID.
During multi-touch, a unique value is assigned to each touch point, allowing identification of the position by this ID.

Return Value
None (void)
Description

Occurs when a finger touching the touch panel moves.
This event occurs when the X and Y coordinates are within this layer.
However, unlike a mouse, it is determined by the current coordinates, so it may not be the same layer where onTouchDown occurred.
