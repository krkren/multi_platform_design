Window.update

Function/Meaning

[Windows*] Force drawing of window content

Type
Window class method
Syntax
update(type=tutNormal)
Arguments
type

Specifies the window drawing type.

  • Specifying tutNormal performs normal drawing (differential drawing),
  • Specifying tutEntire draws the entire window content.
Return Value
None (void)
Description

The argument is ignored in the current version.
tutNormal and tutEntire are not implemented.
