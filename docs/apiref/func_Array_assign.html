Array.assign

Function/Meaning

Copies an array.

Type
Method of Array class
Syntax
assign(src:Array)
Arguments
src

Source Array|Dictionary instance

Return Value
None (void)
Description

Copies the contents of another array specified by the argument.
If a dictionary (Dictionary class object) is specified as the source, the resulting array will contain the dictionary's members alternating between names and values.
The order of members is undefined.
