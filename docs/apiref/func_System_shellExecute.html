System.shellExecute

Function/Meaning

[Windows*] Execute file/program

Type
System class method
Syntax
shellExecute(target, param=)
Arguments
target

Specifies the file or software to execute.
If a file is specified, the program associated with it will be started.

param

Specifies the parameters to pass to the software being executed.
If a file is specified in the target argument, omit this argument or specify an empty string.

Return Value

Returns true if execution is successful, false if it fails.

Description

Executes the file or software specified by target.
Nothing happens on Android.
