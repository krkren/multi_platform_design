VideoOverlay.audioBalance

Function/Meaning
[Windows*] Audio Balance (Panning)
Type
Property of the VideoOverlay class
Description

Represents the pan (sound image position).
You can also set the value.
Specifies the left-right position where the sound is heard.
Specify a value from -100000 to 0 to 100000, where -100000 is fully left, 0 is center, and 100000 is fully right.
When playing a stereo source, panning is achieved by attenuating either the left or right channel (specifying 0 outputs both channels, while -100000 outputs only the left channel).
On Android, this always returns 0. Setting it has no effect.
