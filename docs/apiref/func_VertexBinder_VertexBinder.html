VertexBinder.VertexBinder

Function/Meaning

Constructor

Type
Method of VertexBinder class
Syntax
VertexBinder(vertex:VertexBuffer, stride:int=0, componentCount:int=4, offset:int=0, normalize:bool=false)
Arguments
vertex

The VertexBuffer to associate.

stride

Specifies the stride per vertex in bytes. For example, if passing XY coordinates as float types, it would be sizeof(float)*2, so you would specify 8.

componentCount

Passes the number of vertex components. Specify 2 for XY coordinates, or 4 for RGBA color.

offset

Specifies the offset from the beginning of the VertexBuffer in bytes. Used when a single VertexBuffer stores multiple types of vertex information.

normalize

Specifies whether to normalize. If true, non-float types are normalized to the range [-1.0 to 0.0 to 1.0] before being sent to the shader.

Return Value
None (void)
