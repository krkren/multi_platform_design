Array.join

Function/Meaning

Joins an array into a single string.

Type
Method of the Array class
Syntax
join(delimiter:string, reserve, ignore:bool=false) :string
Arguments
delimiter

Delimiter. Separator character.

reserve

Reserved for future versions.

ignore

If true, void elements in the array are ignored.

Return Value

Returns the joined string.

Description

Joins each element of the array with the delimiter and returns the resulting string.
