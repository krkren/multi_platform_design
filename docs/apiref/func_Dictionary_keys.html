Dictionary.keys

Function/Meaning

Returns an array of the object's keys.

Type
Method of the Dictionary class
Syntax
keys(target:Dictionary) :Array
Arguments
target

The object whose members will be counted

Return Value

An array of keys

Description

Returns an array of the object's keys. This works on objects other than Dictionaries as well.
The order of the returned array is the same as Array.assgin and is not sorted.
