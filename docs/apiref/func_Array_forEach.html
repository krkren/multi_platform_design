Array.forEach

Function/Meaning

Executes the argument function for each element of the array.

Type
Method of the Array class
Syntax
forEach(func, args) :variant
Arguments
func

If a string, the method of that name on the array element is called. If a function, it is called in the form function( value, index, args* ).

args

Specifies any number of arguments to pass to the called function. Can be omitted.

Return Value

The value returned by the called function

Description

Calls the specified function once for each element of the array.
If the context is null, such as with an anonymous function, it is called in the context of the array.
If the function returns a value, processing is interrupted and that value becomes the return value of this function.
When calling a method via a string, processing is not interrupted even if a value is returned, and the method is called for all array elements.
