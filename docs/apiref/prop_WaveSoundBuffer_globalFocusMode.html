WaveSoundBuffer.globalFocusMode

Function/Meaning
Focus Mode
Type
Property of WaveSoundBuffer class
Description

Represents the focus mode.
You can also set the value.
Focus mode determines whether to mute when the application is minimized or becomes inactive.

  • If sgfmNeverMute is specified, it will not mute regardless of the application state.
  • If sgfmMuteOnMinimize is specified, it will mute when the application is minimized.
  • If sgfmMuteOnDeactivate is specified, it will mute when the application is deactivated.

This property exists only on the WaveSoundBuffer class (it does not exist on objects created from WaveSoundBuffer).
When using it, use WaveSoundBuffer.globalFocusMode.
The '-wsmute' (DirectSound mute) setting specified in the command line options takes precedence over this property setting.
On Android, it is muted when it becomes inactive.
In the background, processing speed decreases and normal playback becomes impossible.
