Texture.setDrawSize

Function/Meaning

Setting the drawing reference size

Type
Method of the Texture class
Syntax
setDrawSize(width:int, height:int)
Arguments
width

Reference display width

height

Reference display height

Return Value
None (void)
Description

When drawing at 1:1 scale with Canvas.drawTexture, if nothing is specified, it is drawn at the texture's size. However, if you set the width and height with this method, it will be drawn at the set size when using this texture instance.
Scaling and other operations will also be drawn based on this size.
By setting this when the internal image data size differs from the usual drawing size, you can specify the scaling factor based on the size you want to display.
