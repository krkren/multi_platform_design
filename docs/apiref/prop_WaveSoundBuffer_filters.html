WaveSoundBuffer.filters

Function/Meaning
Filter array
Type
Property of the WaveSoundBuffer class
Description

An array (instance of the Array class) that holds insertion filter objects.
By registering filter objects in this array, you can apply various effects to the audio in real-time during playback.
Changes to the filter array are only applied when the WaveSoundBuffer.open method is executed.
Until then, any changes made to this array will not take effect.
Example:

var buf = new WaveSoundBuffer(window);
(omitted)
buf.filters.clear();
// Clear the filter array
buf.filters.add(new WaveSoundBuffer.PhaseVocoder());
// Add PhaseVocoder filter
buf.filters[0].time = 0.5;
// Double-speed playback
