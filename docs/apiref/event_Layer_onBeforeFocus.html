Layer.onBeforeFocus

Function/Meaning

Immediately before receiving focus

Type
Layer class event
Syntax
onBeforeFocus(layer, blurred, direction)
Arguments
layer

The layer object receiving focus.
Usually 'this' is passed, but if a different layer object is specified when calling the superclass's event of the same name, focus can be passed to that layer.

blurred

The layer object losing focus.

direction

True if focus is changed by a backward focus search, false if done by a forward search.

Return Value
None (void)
Description

This event occurs immediately before receiving focus.
When specifying the layer to receive focus, you can specify that layer in the first argument of the superclass's event of the same name.
