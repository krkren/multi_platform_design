WaveSoundBuffer.labels

Function/Meaning
Labels
Type
Property of the WaveSoundBuffer class
Description

Returns an object representing the labels.
This object is a dictionary where each element's member name corresponds to a label name found in the loop information.
Each element is also a dictionary; the 'name' member represents the label name, the 'position' member represents the label position in milliseconds, and 'samplePosition' represents the label position in samples.
Consider this dictionary to be read-only.
Assigning values or creating new members will have no effect.
Example:


var buf = new WaveSoundBuffer(window);
(omitted)
debug.message(buf.labels['start'].position);
// Position of the label named 'start' in milliseconds
debug.message(buf.labels['start'].samplePosition);
// Position of the label named 'start' in samples
