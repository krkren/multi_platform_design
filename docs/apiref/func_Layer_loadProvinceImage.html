Layer.loadProvinceImage

Function/Meaning

Loading province images

Type
Layer class method
Syntax
loadProvinceImage(image)
Arguments
image

Specifies the image storage to be loaded as the province image.

Return Value
None (void)
Description

Loads the province image for the layer.
Other images remain unchanged.
An exception occurs if the image being loaded is a different size than the layer's image size.
