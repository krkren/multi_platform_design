Canvas.drawMesh

Function/Meaning

Mesh drawing

Type
Method of Canvas class
Syntax
drawMesh(shader:ShaderProgram, count:int, primitiveType:int=VertexBuffer.ptTriangles, offset:int=0)
Arguments
shader

Specifies a shader that has been associated with textures and vertex information.

count

Number of vertices to draw.

primitiveType

Specification of triangles, etc.

offset

The offset within the vertex array to start drawing.

Return Value
None (void)
Description

The mesh (VertexBinder: vertex information) must be associated with the shader (set via properties) before calling this method.
Texture information must also be associated in the same way.
After the call, please clear (=void) the information associated with the shader.
