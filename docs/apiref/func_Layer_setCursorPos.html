Layer.setCursorPos

Function/Meaning

Specifying mouse cursor position

Type
Method of Layer class
Syntax
setCursorPos(x, y)
Arguments
x

Specifies the x-coordinate of the mouse cursor (in this layer's display coordinates) in pixels.
This value can also be set or retrieved using the Layer.cursorX property.

y

Specifies the y-coordinate of the mouse cursor (in this layer's display coordinates) in pixels.
This value can also be set or retrieved using the Layer.cursorX property.

Return Value
None (void)
Description

Specifies the position of the mouse cursor.
