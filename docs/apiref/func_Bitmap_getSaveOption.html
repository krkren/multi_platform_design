Bitmap.getSaveOption

Function/Meaning

Get additional image saving information (1.3.0 or later)

Type
Bitmap class method
Syntax
getSaveOption(type)
Arguments
type

Specifies the image format. The specification method is the same as the save method.

Return Value

Dictionary array of additional information

Description

Gets the additional information that can be specified when saving an image.

Reference
Bitmap.save
