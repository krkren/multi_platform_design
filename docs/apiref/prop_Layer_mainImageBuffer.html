Layer.mainImageBuffer

Function/Meaning
Main image buffer pointer
Type
Property of Layer class
Description

Represents a pointer to the upper-left corner of the main image buffer (a 32bpp bitmap containing color and mask (opacity) information).
This property is intended to provide a means of direct access to the image buffer for plugins and other purposes.
It is returned as an integer type, but should be cast to an appropriate type (such as const unsigned long *) in plugins.
Do not write values to the pointer obtained through this property.
You can write to the pointer if it was obtained via Layer.mainImageBufferForWrite.
If no image is allocated to the layer, NULL (0) is returned.
The image size is represented by the Layer.imageWidth and Layer.imageHeight properties.
Refer to Layer.mainImageBufferPitch for the pointer calculation method.
