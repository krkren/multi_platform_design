Canvas.draw9Patch

Function/Meaning

Draw using 9patch

Type
Method of Canvas class
Syntax
draw9Patch(txture:Texture, width:int, height:int, shader:ShaderProgram=null) :Rect
Arguments
texture

Texture to draw

width

Width to draw

height

Height to draw

shader

Shader to use for drawing. Optional.

Return Value

Returns margin information.

Description

The Texture used for drawing must have 9patch information loaded.
Only the Texture class can be specified.
