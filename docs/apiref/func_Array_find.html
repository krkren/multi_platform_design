Array.find

Function/Meaning

Returns the index where the specified value first appears.

Type
Method of the Array class
Syntax
find(value, offset:int=0) :int
Arguments
value

Search value

offset

Start position

Return Value

The position where it was found. Returns -1 if not found.

Description

Searches for the specified value starting from the index specified by offset and returns the index of the first occurrence.
Returns -1 if the specified value is not found.
Identity comparison (behavior of the === operator) is used for comparison.
