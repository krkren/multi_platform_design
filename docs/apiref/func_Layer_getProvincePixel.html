Layer.getProvincePixel

Function/Meaning

Getting the value of the province image

Type
Layer class method
Syntax
getProvincePixel(x, y)
Arguments
x

Specifies the x-coordinate (in layer image coordinates) to get the value from.

y

Specifies the y-coordinate (in layer image coordinates) to get the value from.

Return Value

Province image value ( 0 - 255 )

Description

Gets the value ( 0 - 255 ) at an arbitrary position in the layer's province image.
An exception will occur if an invalid (out of range) position is specified as image coordinates.
