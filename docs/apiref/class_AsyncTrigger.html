AsyncTrigger

The AsyncTrigger class is a class for firing events at the timing of event delivery immediately after control is returned to KiriKiri.
This function is called an asynchronous trigger.
In a script that adopts an event-driven programming model like KiriKiri, it is useful when you want to perform processing that cannot be executed in an event handler (for example, trying to disable the object that generated the event in that event handler) outside of that event handler.
