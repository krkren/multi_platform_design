WaveSoundBuffer.flags

Function/Meaning
Flags
Type
Property of the WaveSoundBuffer class
Description

Gets an object representing the flags.
This object has properties from 0 to 15, each representing the value of a flag.
Properties can be accessed using the indirect member selection operator ('[ ]' operator).
Values can also be set for these properties.
Values range from 0 to 9999 and cannot be lower or higher than this range.
The count property of this object always returns 16.
This object has a reset method; executing this method resets all flags to 0.
If no media is open, setting values to this object's properties will be ignored.
While this object may look like an array object at first glance, it is not a standard TJS array object (an object of the 'Array' class).
All flags are reset to 0 by the WaveSoundBuffer.open method.
Example:


var buf = new WaveSoundBuffer(window);
(omitted)
buf.flags.reset();
// Reset all flags to 0
var cnt = buf.flags.count;
// cnt will contain 16
buf.flags[4] = 34;
// Assign 34 to flag number 4
