Array.loadStruct

Function/Meaning

Inputs structured data from a file

Type
Method of the Array class
Syntax
loadStruct(storage:string, mode:string=null) :Array
Arguments
storage

The name of the file to load

mode

Specifies the mode string when loading the file.

  • If "o" is followed by a decimal offset, reading starts from that byte position in the file.
Return Value

Returns this Array class object itself if the load is successful.

Description

Loads structured data from a file as an array.
Only files exported in binary format are supported.
