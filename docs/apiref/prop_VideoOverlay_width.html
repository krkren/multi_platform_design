VideoOverlay.width

Function/Meaning
[Windows*] Width of the playback rectangle
Type
Property of the VideoOverlay class
Description

Represents the width of the playback rectangle.
This value can also be set.
In the current version, this property cannot be used during layer playback (the size will always be the same as the video size).
On Android, it always returns 0, and setting it has no effect.
