Array.saveStruct

Function/Meaning

Saves structured data to a file.

Type
Method of the Array class
Syntax
saveStruct(storage:string, mode:string=null) :Array
Arguments
storage

The name of the file to write to.

mode

Specifies the mode string for writing the file.

  • If "c" is specified, simple encryption is applied to the file (it can still be read directly with the load method).
  • If "z" is specified, the file is saved with compression (cannot be used with "c"; it can still be read directly with the load method).
  • If "o" is followed by a decimal offset, writing starts from that byte position in the file.
  • If "b" is specified, the file is saved in binary format (cannot be used with other modes; it can still be read directly with the load method).
Return Value

Returns the Array class object itself.

Description

Outputs the state of the elements to a file in a restorable format.
Each element can contain strings, real numbers, octet sequences, or integers.
If an element is an object, its contents will also be output to the file only if it is an Array or Dictionary class object.
Other types of elements will be output as void.
