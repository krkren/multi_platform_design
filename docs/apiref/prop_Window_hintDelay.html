Window.hintDelay

Function/Meaning
[Windows*] Hint display delay time
Type
Property of the Window class
Description

Represents the hint display delay time in milliseconds.
The value can also be set.
The default is 500 milliseconds.
Setting it to 0 will immediately call Window.onHintChanged (always-on state).
Setting it to -1 will prevent hints from being displayed.

Reference
Window.onHintChanged
