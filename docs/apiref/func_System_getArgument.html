System.getArgument

Function/Meaning

Get command line option

Type
Method of the System class
Syntax
getArgument(name)
Arguments
name

Specifies the name of the command line option to retrieve.
Please prefix it with a '-' (hyphen) (e.g., '-nosplash').

Return Value

Returns the value if the command line option is specified, otherwise returns void.

Description

Command line options must be passed to KiriKiri in the format -name=value or -name.
In the former case, 'value' is returned as the value, and in the latter case, 'yes' is returned as the value.
