VideoOverlay.enabledAudioStream

Function/Meaning
[Windows*] Target audio stream number for playback
Type
Property of the VideoOverlay class
Description

Available only for MPEG files.
Gets the audio stream number targeted for playback.
Returns -1 if no audio stream is found.
Always returns 0 on Android.

Reference
VideoOverlay.selectAudioStream
