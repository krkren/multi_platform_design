Window.close

Function/Meaning

Closes the window

Type
Method of the Window class
Syntax
close()
Arguments
Return Value
None (void)
Description

Closes the window displayed by the Window.showModal method.
Before the window closes, the Window.onCloseQuery event occurs, allowing you to check whether the window can be closed.
On Android, the Window.onCloseQuery event does not occur; instead, Activity.finish() is called to terminate.
