Window.setPos

Function/Meaning

[Windows*] Set window position

Type
Method of Window class
Syntax
setPos(left, top)
Arguments
left

Specifies the left edge position of the window.

top

Specifies the top edge position of the window.

Return Value
None (void)
Description

Specifies the position of the window.
When specifying the window position, it is more efficient to use this method to set both at once rather than setting the Window.left and Window.top properties individually.
