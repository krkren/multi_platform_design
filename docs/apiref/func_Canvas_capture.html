Canvas.capture

Function/Meaning

Captures the entire current drawing content to Bitmap/Texture/Offscreen

Type
Method of the Canvas class
Syntax
capture(dest, front:bool=true)
Arguments
dest

Destination Bitmap/Texture/Offscreen

front

Specifies whether to capture from the front buffer or the back buffer. true for front, false for back.

Return Value
None (void)
Description

The bitmap size is adjusted to the screen size.
For Texture/Offscreen, it is not changed.
