Dictionary.loadStruct

Function/Meaning

Inputs structured data from a file

Type
Method of Dictionary class
Syntax
loadStruct(storage:string, mode:string=null) :Dictionary
Arguments
storage

The name of the file to load

mode

Specifies the mode string when loading the file.

  • If "o" is followed by a decimal offset, reading starts from that byte position in the file.
Return Value

Returns the loaded Dictionary or Array class object.

Description

Loads structured data from a file as a dictionary or an array.
Can load both text and binary formats.
This is a class method.
