Octet.[]

Function/Meaning
'numeric' property
Type
Property of the Octet class
Description

By specifying a number as a property, you can access the octet at that position.
The result is represented as the numeric value of the octet.
For example, if 'oct' is an expression representing an octet sequence, oct[0] is the numeric value of the first octet.
An exception occurs if the index is negative or exceeds the length of the octet sequence.

Assignment is not supported.
