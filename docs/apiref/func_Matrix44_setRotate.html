Matrix44.setRotate

Function/Meaning

Set rotation amount

Type
Method of Matrix44 class
Syntax
setRotate(degree:real, x:real, y:real, z:real)
Arguments
degree

Rotation angle

x

Rotation axis X

y

Rotation axis Y

z

Rotation axis Z

Return Value
None (void)
Description

This only sets the value; no calculations are performed with the current matrix.
Normalization of the rotation axis is recommended.
