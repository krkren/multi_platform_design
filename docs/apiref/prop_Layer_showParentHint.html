Layer.showParentHint

Function/Meaning
Whether to inherit parent layer hints
Type
Property of the Layer class
Description

Indicates whether to inherit the hint from the parent layer.
You can also set the value.
If true, and the Layer.hint property is an empty string, it searches up the parent layers and displays the hint from the first layer found that has a hint set.
If the Layer.hint property is not an empty string, it displays that hint.
If false, it displays the Layer.hint property if it is not an empty string, and displays no hint if it is an empty string.
