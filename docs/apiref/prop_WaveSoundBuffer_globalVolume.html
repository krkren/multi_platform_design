WaveSoundBuffer.globalVolume

Function/Meaning
Global volume
Type
Property of the WaveSoundBuffer class
Description

Represents the global volume (master volume).
The value can also be set.
This volume affects all WaveSoundBuffer instances.
Specify a value from 0 to 100000, where 0 is completely muted and 100000 is 100% volume.
The default value is 100000.
This property only exists on the WaveSoundBuffer class (this property does not exist on objects created from WaveSoundBuffer).
When using it, please use WaveSoundBuffer.globalVolume.
