Layer.assignImages

Function/Meaning

Copying images

Type
Layer class method
Syntax
assignImages(src)
Arguments
src

Specifies the source layer.

Return Value
None (void)
Description

Copies all main, mask, and province images from the layer specified by src.
The image size will be the same as the source layer's image size.
Other information is not copied.
Although it is called a copy, in reality, it simply results in the same image being shared by two or more layers, so this method takes almost no execution time.
