Array.remove

Function/Meaning

Removes elements with the same value as the specified value.

Type
Method of the Array class
Syntax
remove(value, all:bool=true)
Arguments
value

Value to remove

all

Whether to target all occurrences or not

Return Value
None (void)
Description

Performs an identity comparison (behavior of the === operator) between the specified value and each element, and removes matching elements.
If "all" is false, only the first element found is removed.
Otherwise, all matching elements found are removed.
