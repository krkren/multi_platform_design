WaveSoundBuffer.fade

Function/Meaning

Starts fading

Type
Method of the WaveSoundBuffer class
Syntax
fade(to, time, delay=0)
Arguments
to

Specifies the target volume.
Refer to the WaveSoundBuffer.volume property for volume specification.

time

Specifies the time for the fade in ms.

delay

Specifies the delay before the fade starts in ms.

Return Value
None (void)
Description

Starts a fade (continuous change in volume).
