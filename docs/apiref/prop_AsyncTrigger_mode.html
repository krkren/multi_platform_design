AsyncTrigger.mode

Function/Meaning
Mode
Type
Property of the AsyncTrigger class
Description

Represents the operation mode.
The value can also be set.
Specify one of the following values.
atmNormal : Fired during the normal event delivery stage.
atmExclusive : Fired with priority over other asynchronous events
atmAtIdle : Fired when in an idle state (when there are no more events to deliver).
When triggers are pulled simultaneously, the firing order is atmExclusive, atmNormal, atmIdle.
If multiple triggers of the same mode are pulled, they are fired in the order they were pulled.
