Dictionary.forEach

Function/Meaning

Executes the argument function for each element of the argument dictionary.

Type
Method of the Dictionary class
Syntax
forEach(target:Dictionary, func, args) :variant
Arguments
target

Dictionary

func

If a string is specified, the method with that name on the element is called. If a function is specified, it is called in the form function( value, key, args* ).

args

Specifies any number of arguments to pass to the function. Can be omitted.

Return Value

The value returned by the called function

Description

Calls the specified function once for each element of the dictionary.
If the context is null (such as with an anonymous function), it is called in the context of the dictionary.
If the function returns a value, processing is interrupted and that value becomes the return value of this function.
When a method is called by string, processing is not interrupted even if a value is returned, and the method is called for all elements.
