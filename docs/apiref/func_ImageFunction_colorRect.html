ImageFunction.colorRect

Function/Meaning

Rectangular semi-transparent fill

Type
Method of the ImageFunction class
Syntax
colorRect(bmp, value, opa=255, rect=null, face=dfAlpha, cliprect=null)
Arguments
bmp

Specifies the Bitmap object to be filled.

value

Specifies the color or value to fill with.
The meaning of this value changes depending on the face argument.

  • dfAlpha : Specify the color in 0xRRGGBB format
  • dfAddAlpha : Specify the color in 0xRRGGBB format
  • dfOpaque : Specify the color in 0xRRGGBB format
  • dfMask : Specify the mask (opacity) value ( 0 - 255 )

If dfOpaque is specified, mask information is ignored.
Also, if dfMask is specified, the color information remains as is.
In the case of dfAlpha, if opa is negative, this argument is ignored.

opa

Specifies the opacity to fill with ( -255 - 0 - 255 ).
This argument is ignored if the face value is dfMask or dfProvince (always fully opaque).
Specifying a negative number is only valid when face is dfAlpha; in this case, the value argument is ignored, and opacity is removed from the image (specifying -255 makes the rectangle completely transparent).

rect

Specifies the rectangle to be filled as a Rect object in pixels (relative to the image position).
If unspecified, the entire area is targeted.

face

Specifies the drawing method.

  • If dfAlpha is specified, the image is treated as an image with an alpha channel and drawn.
  • If dfAddAlpha is specified, the image is treated as an image with an additive alpha channel and drawn.
  • If dfOpaque is specified, the image is treated as fully opaque and drawn.
cliprect

Specifies the clipping region as a Rect object (relative to the Bitmap's image position).
If unspecified, the entire area is targeted.

Return Value
None (void)
Description

Fills the rectangle of the specified Bitmap image using the specified method.
Unlike ImageFunction.fillRect, you can specify transparency to fill with semi-transparency.
