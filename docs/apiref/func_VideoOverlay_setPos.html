VideoOverlay.setPos

Function/Meaning

[Windows*] Specifies the top-left position of the playback rectangle

Type
VideoOverlay class method
Syntax
setPos(left, top)
Arguments
left

Specifies the left edge position of the rectangular area for media playback.

top

Specifies the top edge position of the rectangular area for media playback.

Return Value
None (void)
Description

Specifies the top-left position of the rectangular area for media playback.
Coordinates are specified in pixels within the window's client area (the area where layers can be displayed).
