VertexBuffer.VertexBuffer

Function/Meaning

Creates an uninitialized vertex buffer

Type
Method of the VertexBuffer class
Syntax
VertexBuffer(size:int, dataType:int, updateType:int, isIndex:bool=false)
Arguments
size

Buffer size in bytes

dataType

Data type. dtByte/dtUByte/dtShort/dtUShort/dtInt/dtFloat

updateType

Data update frequency. utStream/utStatic/utDynamic

isIndex

Whether it is an index buffer. false for a vertex buffer

Return Value
None (void)
Description
