Storages.extractStorageName

Function/Meaning

Extract storage name

Type
Method of Storages class
Syntax
extractStorageName(storage)
Arguments
storage

Specifies the storage name from which you want to extract the storage name part.

Return Value

Returns the storage name.
Returns an empty string if there is no storage name.

Description

Extracts and returns the storage name part (excluding the path) from the specified storage name.
For example, if "System/hoge.txt" is passed, "hoge.txt" is returned.
