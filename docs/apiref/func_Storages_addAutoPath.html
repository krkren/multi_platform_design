Storages.addAutoPath

Function/Meaning

Add to auto-search path

Type
Method of the Storages class
Syntax
addAutoPath(path)
Arguments
path

Specifies the path to add to the auto-search path.
The end of the path must end with '>' when specifying the root folder within an archive, or '/' when specifying a normal folder.
(Example: "Archive/arc.xp3>" or "System/"). Since 2.19 beta 14, the archive delimiter has changed from '#' to '>'.

Return Value
None (void)
Description

Adds the specified path to the auto-search path.
When searching for available storage, Kirikiri looks through the paths registered in this auto-search path.
Paths added later to the auto-search path are given higher priority during the search.
Additionally, the project folder is always searched with the highest priority.
