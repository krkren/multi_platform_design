Window.startDrawing

Function/Meaning

Explicitly call drawing.

Type
Method of the Window class
Syntax
startDrawing()
Arguments
Return Value
None (void)
Description

This method indirectly calls onDraw.
If you want to perform drawing yourself, call this method and draw within onDraw.
Please try to avoid calling this method explicitly; instead, specify Window.drawCycle to perform periodic drawing.
This may not be supported on other platforms in the future.
