Layer.absoluteOrderMode

Function/Meaning
Whether in absolute position mode
Type
Property of the Layer class
Description

Represents the stacking order mode of direct child layers.
This value can also be set.
If set to false, relative positioning is used, and the Layer.order property represents the order.
If set to true, absolute positioning is used, and the Layer.absolute property represents the order.
