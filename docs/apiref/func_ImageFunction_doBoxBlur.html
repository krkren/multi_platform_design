ImageFunction.doBoxBlur

Function/Meaning

Apply box blur

Type
Method of ImageFunction class
Syntax
doBoxBlur(bmp, xblur=1, yblur=1, cliprect=null, isalpha=true)
Arguments
bmp

Specifies the Bitmap object to apply the box blur to.

xblur

Specifies the horizontal blur range.

yblur

Specifies the vertical blur range.

cliprect

Specifies the clipping region as a Rect object (in Bitmap coordinates).
If unspecified, the entire image is targeted.

isalpha

Specifies whether to process the alpha channel.

Return Value
None (void)
Description

Applies a blur effect.
The algorithm used is "box blur".
A box blur calculates the average luminance of pixels within a "range" defined by the two parameters xblur and yblur.
For example, if xblur=10 and yblur=2, the average luminance of pixels in a rectangular range of -10 to 10 horizontally and -2 to 2 vertically around the target pixel is calculated, and that becomes the final luminance of that pixel.
The area of the range can be calculated as (xblur*2+1) * (yblur*2+1).
In the current version, if this area is less than 256, a faster algorithm is used compared to when the area is 256 or more.
If isalpha is true, a special algorithm for alpha blending is used, which is slightly slower.
If false, a faster algorithm is used.
