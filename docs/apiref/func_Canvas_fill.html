Canvas.fill

Function/Meaning

Fill with color

Type
Method of Canvas class
Syntax
fill(width:int, height:int, colors=-1, shader=null)
Arguments
width

Width of the fill area

height

Height of the fill area

colors

Vertex colors (ARGB) for the 4 vertices. If a single numeric value, it is a solid color. If an array, 4 vertices are specified individually.

shader

Fill shader. When null, it is filled using defaultFillShader.

Return Value
None (void)
Description

XY coordinates are specified via the matrix.
