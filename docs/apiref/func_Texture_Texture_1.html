Texture.Texture

Function/Meaning

Mipmap generation constructor

Type
Method of the Texture class
Syntax
Texture(sizelist:Array, filename:string, type=stFastAreaAvg, typeopt:real=1)
Arguments
sizelist

Specifies the sizes for generating mipmaps as an array of width and height pairs.

filename

Image filename

type

Specifies the reduction algorithm. Same as the type argument for ImageFunction.operateStretch or Layer.operateStretch.

typeopt

Sharpness for cubic interpolation. Currently has no effect for other interpolation methods.

Return Value
None (void)
