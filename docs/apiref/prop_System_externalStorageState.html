System.externalStorageState

Function/Meaning
[Android+]External storage state[r]
Type
Property of the System class
Description

Value obtained via Environment.getExternalStorageState().
One of the following:

  • "unknown" Unknown state
  • "removed" Removed state
  • "unmounted" Unmounted state
  • "checking" Checking disk
  • "nofs" Unformatted or unsupported file system
  • "mounted" Mounted and read/write accessible
  • "mounted_ro" Mounted as read-only
  • "shared" Not mounted, such as USB mass storage
  • "bad_removal" Removed before being unmounted
  • "unmountable" Cannot be mounted
