Window.setSize

Function/Meaning

[Windows*] Set window size

Type
Method of Window class
Syntax
setSize(width, height)
Arguments
width

Specifies the width of the window.

height

Specifies the height of the window.

Return Value
None (void)
Description

Specifies the size of the window.
When specifying the window size, it is more efficient to use this method to set both at once rather than setting the Window.width and Window.height properties individually.
