VideoOverlay.setSegmentLoop

Function/Meaning

[Windows*] Set inter-frame loop

Type
VideoOverlay class method
Syntax
setSegmentLoop(comeFrame, goFrame)
Arguments
comeFrame

Loop destination frame (loop start frame).
When playback reaches goFrame, the playhead moves to this frame.

goFrame

Loop end frame.
When the display of the frame immediately preceding this frame ends, the playhead moves to comeFrame.

Return Value
None (void)
Description

Performs loop processing between the specified frames.
At the loop end (goFrame), an onPeriod event occurs.
A keyframe must be set at the comeFrame in the movie file.
If not set, the playback position will move to the keyframe closest to the specified frame when moving from the loop end to the start.
