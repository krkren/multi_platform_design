Timer.Timer

Function/Meaning

Constructing Timer object

Type
Timer class method
Syntax
Timer(owner, actionname=action)
Arguments
owner

Specifies the event target.

actionname

Specifies the name of the method to receive events in the target object specified by owner.
If an empty string is specified, owner is treated as a method and is called directly at each event interval.

Return Value
None (void)
Description

Constructs an object of the Timer class.
In the initial state, the interval property is 1000 and the enabled property is false.
