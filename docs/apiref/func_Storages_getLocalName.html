Storages.getLocalName

Function/Meaning

Get local filename

Type
Method of Storages class
Syntax
getLocalName(path)
Arguments
path

Specifies the unified storage name to be converted to a local filename.

Return Value

Returns the local filename.
If it cannot be converted to a local filename, an empty string is returned or an exception is raised.

Description

Converts the specified unified storage name into the OS native format (local filename) and returns it.
