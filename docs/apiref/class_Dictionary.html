Dictionary

The Dictionary class is a class that handles dictionary arrays.

Dictionary arrays, also known as associative arrays, are objects that hold multiple pairs of names and their associated values.
Like arrays, they are accessed using the [ ] (indirect reference) operator, but instead of an index, a string is used as the name.
If the name is valid as an identifier, the . (direct reference) operator can also be used.
Also, referencing a non-existent name will result in void.
To delete an element, use the delete operator.

Dictionary Array in Expression

Using %[ ], you can describe a Dictionary class object in an expression on the spot.
List the expressions that will be the initial elements, separated by commas, in the form element name => element value. For example,


var dic = %["a" => 1, "b" => 2, "c" => 3];

If you write as above, dic will be assigned a reference to a Dictionary class object where the element name and element pairs are "a":1, "b":2, and "c":3 respectively.
Internally, => is exactly the same as a comma, but for readability, => can be used (same as perl).

Accessing Dictionary Class Methods

Dictionary class objects do not have any members when they are created.
For example, if you try to use the assign method and write dict.assign(src) for a Dictionary class object dict, it will result in an error because dict does not have a method called assign.
Therefore, use the incontextof operator to execute methods that belong directly to the Dictionary class with the target Dictionary class object as the context.

Example:

var a = %[];
var b = %[];
(Dictionary.assign incontextof a)(b); // Copy b to a
(Dictionary.clear incontextof b)(); // Clear the contents of b
