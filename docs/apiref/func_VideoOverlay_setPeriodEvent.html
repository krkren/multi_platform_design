VideoOverlay.setPeriodEvent

Function/Meaning

[Windows*] Specifies an event to occur at a specific frame

Type
VideoOverlay class method
Syntax
setPeriodEvent(eventFrame)
Arguments
eventFrame

Specifies the frame at which to trigger the onPeriod event.

Return Value
None (void)
Description

Triggers the onPeriod event at the specified frame.
The onPeriod event is cleared once it occurs.
If you want it to occur again, please set it again using this method.

Reference
VideoOverlay.cancelPeriodEvent
