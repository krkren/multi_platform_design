Layer.setMainPixel

Function/Meaning

Setting the main image color

Type
Method of the Layer class
Syntax
setMainPixel(x, y, color)
Arguments
x

Specifies the x-coordinate (in layer image coordinates) where the color will be set.

y

Specifies the y-coordinate (in layer image coordinates) where the color will be set.

color

Specifies the color to set in 0xRRGGBB format.

Return Value
None (void)
Description

Sets the color at an arbitrary position in the layer's main image (the image holding the color data).
An exception will occur if a position that is invalid (out of range) for the image coordinates is specified.
