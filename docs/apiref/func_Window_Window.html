Window.Window

Function/Meaning

Window object construction

Type
Window class method
Syntax
Window(parent)
Arguments
parent

Specifies the parent window.
If a parent window is specified, it will be created as a child window.
Omit if not specifying.
On Android, only one Window can be created, so this parameter cannot be used.

Return Value
None (void)
Description

Constructs a Window class object.
The window is created in a hidden state, and its position and size are undefined (a random size at some position).
On Android, only one can be created, it operates in full screen, and position or size settings are ignored.
