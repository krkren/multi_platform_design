Layer.type

Function/Meaning
Layer display type
Type
Property of Layer class
Description

Represents the display type of the layer.
You can also set the value.

  • Specifying ltOpaque or ltCoverRect disables per-pixel alpha blending. ltCoverRect and ltOpaque have the same meaning. If the Layer.opacity property is 255, it will be displayed as a completely opaque rectangle. The mask image is ignored. The drawing method (specified by Layer.face) suitable for this type is dfOpaque.
  • Specifying ltAlpha or ltTransparent enables per-pixel alpha blending. ltTransparent and ltAlpha have the same meaning. The mask image is used for transparency. The drawing method suitable for this type is dfAlpha.
  • Specifying ltAddAlpha enables per-pixel additive alpha blending. The drawing method suitable for this type is dfAddAlpha.
  • Specifying ltAdditive performs additive blending. The mask image is ignored. The drawing method suitable for this type is dfOpaque.
  • Specifying ltSubtractive performs subtractive blending. The mask image is ignored. The drawing method suitable for this type is dfOpaque.
  • Specifying ltMultiplicative performs multiplicative blending. The mask image is ignored. The drawing method suitable for this type is dfOpaque.
  • Specifying ltDodge performs dodge blending. The mask image is ignored. The drawing method suitable for this type is dfOpaque.
  • Specifying ltDarken performs darken (minimum) blending. The mask image is ignored. The drawing method suitable for this type is dfOpaque.
  • Specifying ltLighten performs lighten (maximum) blending. The mask image is ignored. The drawing method suitable for this type is dfOpaque.
  • Specifying ltScreen performs screen blending. The mask image is ignored. The drawing method suitable for this type is dfOpaque.

Please refer to the Graphic System for other layer display types.
