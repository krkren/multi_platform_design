WaveSoundBuffer.volume2

Function/Meaning
Secondary Volume
Type
Property of the WaveSoundBuffer class
Description

Represents the playback volume. This value can be set.
Unlike the WaveSoundBuffer.volume property, this property is not changed by the WaveSoundBuffer.fade method.
The final volume is determined by the product of the volume property and this property.
If the volume property is 100000 (100%) and the volume2 property is also 100000 (100%), it will play at 100% x 100% = 100% volume.
If the volume property is 50000 (50%) and the volume2 property is 75000 (75%), it will play at 50% x 75% = 37.5% volume.
