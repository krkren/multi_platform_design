Layer.setAttentionPos

Function/Meaning

Specifying the attention position

Type
Method of Layer class
Syntax
setAttentionPos(left, top)
Arguments
left

Specifies the x-coordinate (in this layer's display coordinates) to focus on, in pixels.
This value can also be set or retrieved via the Layer.attentionLeft property.

top

Specifies the y-coordinate (in this layer's display coordinates) to focus on, in pixels.
This value can also be set or retrieved via the Layer.attentionTop property.

Return Value
None (void)
Description

Specifies the attention position.
The attention position is usually set to the position of the caret (the blinking bar in text editors, etc., that indicates the character input position from the keyboard).
IME composition strings (unconverted characters) are displayed at this attention position.

Reference
