Layer.update

Function/Meaning

Update image

Type
Method of Layer class
Syntax
update(left, top, width, height)
Arguments
left

Specifies the left position of the rectangle to update, in pixels, using display coordinates.

top

Specifies the top position of the rectangle to update, in pixels, using display coordinates.

width

Specifies the width of the rectangle to update, in pixels, using display coordinates.

height

Specifies the height of the rectangle to update, in pixels, using display coordinates.

Return Value
None (void)
Description

This method forces the layer to be drawn to the screen.
While screen updates occur automatically when drawing methods are executed, this method can be used to force a screen redraw.
If all arguments are omitted, the entire layer will be redrawn.
This method sets the Layer.callOnPaint property to true.
Until the screen is actually drawn, screen drawing will only occur once regardless of how many times this method is called (it is cached).
