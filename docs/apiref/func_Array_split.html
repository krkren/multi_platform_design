Array.split

Function/Meaning

Splits a string.

Type
Method of the Array class
Syntax
split(pattern:string, text:string, reserve, ignoreSpace:bool=false) :Array
Arguments
pattern

Specifies a pattern or a set of delimiters.
The pattern must be a regular expression object.
The delimiter set specifies characters to be used as separators.
Multiple delimiters can be specified by providing a string.

text

Specifies the string to be split.

reserve

Reserved for future versions.

ignoreSpace

Specify true if you do not want to copy empty elements to the array (for example, when using commas as delimiters and encountering ,,,,,).

Return Value

Returns this Array class object itself.

Description

Initializes the array and copies the string split by the pattern or delimiter set into this array.
