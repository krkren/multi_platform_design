Layer.mainImageBufferForWrite

Function/Meaning
Main image buffer pointer (for writing)
Type
Property of the Layer class
Description

Represents a pointer to the upper-left corner of the main image buffer (a 32bpp bitmap containing color and mask (opacity) information).
This property is intended to provide a means of direct access to the image buffer for plugins and similar purposes.
It is returned as an integer type, but in plugins, it should be cast to an appropriate type (such as unsigned long *).
Unlike Layer.mainImageBuffer, values can be written to the pointer obtained through this property.
Internally, KiriKiri shares identical images between multiple layers, but accessing this property will break that sharing state (copy-on-write).
If no image is assigned to the layer, NULL (0) is returned.
The image size is represented by the Layer.imageWidth and Layer.imageHeight properties.
Refer to Layer.mainImageBufferPitch for how to calculate the pointer address.
