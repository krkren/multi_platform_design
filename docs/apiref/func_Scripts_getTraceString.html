Scripts.getTraceString

Function/Meaning

Get call history

Type
Method of the Scripts class
Syntax
getTraceString(limit=0)
Arguments
limit

Specifies the maximum call depth for retrieving history.
If this argument is omitted or 0 is specified, it retrieves as much history as possible.

Return Value

The call history as a string

Description

Retrieves the method call history as a string.
You can obtain the history at the point this method is called.
To use this method, the -debug (debug mode) command line option must be enabled.
If debug mode is disabled, this method returns an empty string.
The returned string looks like, for example:
'messagelayer.tjs(1561)[(function) addButton] <-- mainwindow.tjs(4463)[(function expression) (anonymous)] <-- conductor.tjs(427)[(function) onTag] <-- conductor.tjs(95)[(function) timerCallback]'
.
