Layer.onKeyPress

Function/Meaning

Character input

Type
Event of the Layer class
Syntax
onKeyPress(key, process=true)
Arguments
key

The character that was input.

process

When calling the parent class's onKeyPress, if you pass false for this argument, the parent class will not perform default processing (such as moving focus) for that key.

Return Value
None (void)
Description

Occurs when a character is input.
Unlike Layer.onKeyDown, which handles virtual key codes, this event handles the character itself.
This event does not occur if the pressed key is not related to a character (such as function keys).
