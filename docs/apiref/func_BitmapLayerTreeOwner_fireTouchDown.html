BitmapLayerTreeOwner.fireTouchDown

Function/Meaning

Notifies the layer that a touch occurred

Type
Method of the BitmapLayerTreeOwner class
Syntax
fireTouchDown(x, y, cx, cy, id)
Arguments
x

The x-coordinate of the touched position (client coordinates)

y

The y-coordinate of the touched position (client coordinates)

cx

The number of horizontal pixels the finger is in contact with

cy

The number of vertical pixels the finger is in contact with

id

Touch ID

Return Value
None (void)
Description
