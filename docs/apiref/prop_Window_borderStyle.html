Window.borderStyle

Function/Meaning
[Windows*] Window Appearance
Type
Property of the Window class
Description

Represents the appearance of the window. You can also set this value.
On Android, it is always bsNone.
The following values can be set:

  • bsDialog : Non-resizable, with an appearance similar to a dialog box.
  • bsSingle : A non-resizable window.
  • bsNone : A window with no border.
  • bsSizeable : A standard resizable window. This is the default.
  • bsToolWindow : A non-resizable tool window (a window with a small caption).
  • bsSizeToolWin : Similar to bsToolWindow, but resizable.
