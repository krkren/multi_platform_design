String.toUpperCase

Function/Meaning

Returns a string with lowercase alphabetic characters converted to uppercase

Type
String class method
Syntax
toUpperCase() :string
Arguments
Return Value

Converted string

Description

Returns a string with lowercase alphabetic characters converted to uppercase.
Full-width characters are not converted.
This method does not affect the original string.
In the current version, conversion is only performed for characters with character codes less than 0x80.
