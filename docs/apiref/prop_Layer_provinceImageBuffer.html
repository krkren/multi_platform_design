Layer.provinceImageBuffer

Function/Meaning
Province image buffer pointer
Type
Property of the Layer class
Description

Represents a pointer to the upper-left corner of the province image buffer (an 8bpp bitmap containing province information).
This property is intended to provide a means of direct access to the image buffer for plugins and other purposes.
It is returned as an integer type, but should be cast to an appropriate type (such as const unsigned char * etc.) for use in plugins.
Do not write values to the pointer obtained through this property.
You can write to the pointer if it was obtained through Layer.provinceImageBufferForWrite.
If no image is allocated, NULL (0) is returned.
If no image is allocated, the entire area must be considered as province number 0.
The size of the image is represented by the Layer.imageWidth and Layer.imageHeight properties.
Refer to Layer.provinceImageBufferPitch for the pointer calculation method.
