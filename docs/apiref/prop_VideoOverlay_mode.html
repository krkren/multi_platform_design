VideoOverlay.mode

Function/Meaning
[Windows*] Specify overlay or layer rendering
Type
Property of the VideoOverlay class
Description

Indicates whether it is in overlay mode, layer rendering mode, or mixer mode.
The value can also be set.
It will be vomOverlay for overlay mode, vomLayer for layer rendering mode, vomMixer for mixer mode, and vomMFEVR for Media Foundation mode.
There is no playback mode on Android.
